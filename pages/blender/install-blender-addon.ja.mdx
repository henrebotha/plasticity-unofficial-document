# Blenderアドオンのインストール

Blender Bridgeの機能を使うには、Blenderへのアドオンのインストールと、Plasticityでの機能の有効化が必要です。

## Blenderアドオンのインストール

1. 公式のGitHubリポジトリからBlender Bridgeアドオンをダウンロードします。以下は最新バージョンへの直接のリンクです。: 
    [plasticity-blender-addon-main.zip](https://github.com/nkallen/plasticity-blender-addon/archive/refs/heads/main.zip)

2. Blenderを起動します。

3. 通常の方法でplasticity-blender-addon-main.zipをBlenderアドオンとしてインストールします。
    - Blenderを開きます。
    - Edit > Preferences > Add-ons に移動します。
    - "Install..." をクリックし、ダウンロードしたzipファイルを選択します。
    - "[ ]Object: Plasticity" がリストに表示されるまでに時間がかかる場合があることに注意してください。

4. インストールが完了すると、3Dビューポートのサイドバーに"Plasticity" を見つけることができます。

## Plasticityでの機能の有効化

1. Plasticityを起動します。

2. 画面左上のPメニューからPreferencesを開きます。

3. 左側のServerを開き、Enabledにチェックリストを入れます。

以上で、Blender Bridgeの機能を使う準備が整いました。
