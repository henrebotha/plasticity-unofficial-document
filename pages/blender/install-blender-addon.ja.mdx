# Install Blender Add-on

1. Download the Blender Bridge add-on from the official GitHub repository. Here is the direct link to the latest version:
    [plasticity-blender-addon-main.zip](https://github.com/nkallen/plasticity-blender-addon/archive/refs/heads/main.zip)

2. Install plasticity-blender-addon-main.zip as a Blender add-on normally.
    - Open Blender.
    - Navigate to Edit > Preferences > Add-ons.
    - Click on "Install..." and select the downloaded zip file.
    - Please note that it may take some time for "[ ]Object: Plasticity" to appear in the list.

3. Once installed, you can find "Plasticity" in the sidebar of the 3D Viewport.






# Blenderアドオンのインストール

1. 公式のGitHubリポジトリからBlender Bridgeアドオンをダウンロードします。以下は最新バージョンへの直接のリンクです。: 
    [plasticity-blender-addon-main.zip](https://github.com/nkallen/plasticity-blender-addon/archive/refs/heads/main.zip)

2. 通常の方法でplasticity-blender-addon-main.zipをBlenderアドオンとしてインストールします。
    - Blenderを開きます。
    - Edit > Preferences > Add-ons に移動します。
    - "Install..." をクリックし、ダウンロードしたzipファイルを選択します。
    - "[ ]Object: Plasticity" がリストに表示されるまでに時間がかかる場合があることに注意してください。

3. インストールが完了すると、3Dビューポートのサイドバーに"Plasticity" を見つけることができます。
