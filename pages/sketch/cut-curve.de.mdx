# Cut Curve

<Reference shortcut="C" command="Cut Curve" button="Links von der Befehlsleiste" />

Schneiden Sie die Ziel-Curve mit einer Schneide-Curve oder Face. Wenn eine Curve der Schneide dient, wird der Schnitt basierend auf einer von dieser Curve erzeugten Oberfläche gemacht.

1. Drücken Sie <kbd>C</kbd> oder klicken Sie auf die Schaltfläche <ButtonIcon type="button-cut" />.
2. Drücken Sie erneut <kbd>C</kbd>, um zum **Cut Curve** Befehl zu wechseln.
3. Legen Sie das Ziel und den Schneide fest.
   - <Kst>Befehlsdialog</Kst>
   - **Select target bodies**: Wählen Sie die Curve aus, die geschnitten werden soll.
   - **Select cutters**: Wählen Sie die Curve oder Face aus, die als Schneide verwendet werden soll.
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>S</kbd> **Screen space**: Legt die Schnittrichtung so fest, dass sie mit der sichtbaren Richtung übereinstimmt.
4. Bestätigen Sie mit OK im Befehlsdialog oder klicken Sie mit der rechten Maustaste.
