# Überflüssige Topologie löschen

<Reference shortcut="" command="Delete Redundant Topology" button="" />


Dieser Befehl ruft automatisch den folgenden Befehl auf, abhängig vom ausgewählten Objekt:
    - Wenn Curves ausgewählt sind: **Delete Redundant Topology (Curve)**, unten beschrieben.
    - Wenn Solids oder Sheets ausgewählt sind: siehe [**Delete Redundant Topology (Solid)**](/solid/delete-redundant-topology#delete-redundant-topology-solid) im Abschnitt Solid-Befehle.

## Überflüssige Topologie löschen (Curve)[#delete-redundant-topology-curve]

Sie können überflüssige Control Vertices(CVs) von Curves löschen.

1. Wählen Sie die Curves aus, um überflüssige CVs zu löschen.
2. Verwenden Sie den **Delete Redundant Topology** Befehl in der Befehlspalette.
3. Die überflüssigen CVs werden gelöscht.
