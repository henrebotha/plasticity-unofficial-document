# Offset Curve

<Reference shortcut="O" command="Offset Curve" button="Right of Command Bar" />

This command automatically calls the following command depending on the selected object.
   - When Curves are selected: [**Offset Planar Curve**](#offset-planar-curve) described below.
   - When Regions are selected: [**Offset Region**](#offset-region) described below.
   - When Faces are selected: see [**Offset Face Loop**](../solid/offset-curve#offset-face-loop) of **Offset Curve** command  in the Solid Commands section.
   - When Edges are selected: see [**Offset Edge**](../solid/offset-curve#offset-edge) of **Offset Curve** command  in the Solid Commands section.

## Offset Planar Curve

You can offset planar Curves to create new Curves.

1. Select the Curves you want to offset.
2. Press <kbd>O</kbd>.
3. Move the yellow arrow to specify the distance.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - **Distance:** Specify the Distance.
   - **Gap fill:** 
        - **Round:** Rounds convex corners. 
        - **Linear:** Connects convex corners with straight lines. 
        - **Natural:** Natural
   - <kbd>Tab</kbd> **Lock distances:** Offset with equal distance on both sides.
   - <kbd>F</kbd> **Freestyle:** It functions similarly to the [**Freestyle Offset Planar Curve**](#freestyle-offset-planar-curve) described below.
   - <kbd>D</kbd> **Distance:** Specify the distance with mouse movement.
5. Confirm with OK in the Command Dialog or right-click.

## Freestyle Offset Planar Curve

You can offset planar Curves with snap grid or snap object to create new Curves.

This command can be executed not only through the **Freestyle** option in the **Offset Curve** command but also by using the **Freestyle Offset Planar Curve** in the Command Palette.

1. Select the Curves you want to offset.
2. Press <kbd>O</kbd>.
3. Press <kbd>F</kbd> to enter Freestyle mode.
4. Move the cursor to the offset point and click.
   - If Grid Snaps are enabled, the offset point snaps to the grid.
   - If Object Snaps are enabled, the offset point snaps to the object.
   - **Distance:** Specify the Distance.
   - **Gap fill:** 
        - **Round:** Rounds convex corners. 
        - **Linear:** Connects convex corners with straight lines. 
        - **Natural:** Natural    
   - <kbd>Tab</kbd> **Lock distances:** Offset with equal distance on both sides.
5. Confirm with OK in the Command Dialog or right-click.

## Offset Region

You can offset Regions to create new Regions.

1. Select the Regions you want to offset.
2. Press <kbd>O</kbd>.
3. Drag the yellow arrow to specify the distance.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - **Distance:** Specify the Distance.
   - **Gap fill:** 
        - **Round:** Rounds convex corners. 
        - **Linear:** Connects convex corners with straight lines. 
        - **Natural:** Natural    
   - <kbd>Tab</kbd> **Lock distances:** Offset with equal distance on both sides.
   - <kbd>I</kbd> **Individual:** When multiple Regions are selected, turning on this option offsets the Regions in a combined form.
   - <kbd>D</kbd> **Distance:** Specify the distance with mouse movement.
5. Confirm with OK in the Command Dialog or right-click.
