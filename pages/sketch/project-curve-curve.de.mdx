# Project Curve Curve [#project-curve-curve]

<Reference shortcut="I" command="Project Curve Curve" button="Rechts von der Befehlsleiste" />

Erstellen Sie eine neue Kurve, indem Sie die Schnittpunkte von Oberflächen projizieren, die aus zwei unabhängigen Kurven erzeugt wurden.

## Grundlegende Nutzung

1. Wählen Sie die beiden Kurven aus, die projiziert werden sollen.
2. Drücken Sie <kbd>I</kbd> oder klicken Sie auf die Schaltfläche.
3. Die projizierte Schnittkurve wird erstellt.
