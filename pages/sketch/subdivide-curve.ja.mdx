# Subdivide Curve（カーブの制御点を増やす）

<Reference shortcut="Shift + S" command="Subdivide Curve" button="コマンドバー右" />

カーブの制御点を増やすことができます。

1. 制御点を増やしたいカーブを選択します。

2. <kbd>Shift + S</kbd>を押すか、コマンドバー右の<ButtonIcon type="button-subdivide-curve" />をクリックします。

3. 制御点の数が増え、増えた制御点が選択されます。
