# Alternative Duplicate

<Reference shortcut="Alt + D" command="Alternative Duplicate" button="" />

Projizieren Sie die Curves auf die aktuelle Bauebene entlang der normalen Richtung, wodurch eine Curves erzeugt wird.

## Grundlegende Nutzung

1. Legen Sie die gewünschte Bauebene für die Projektion fest.
2. Wählen Sie die Curves aus, auf die projiziert werden soll.
3. Drücken Sie <kbd>Alt + D</kbd>, um den Befehl auszuführen.
4. Die projizierten Curves werden auf der aktuellen Bauebene erstellt.
5. Optional können Sie die Curves mit dem Move Gizmo verschieben.
6. Klicken Sie mit der rechten Maustaste oder klicken Sie auf OK im Befehlsdialog, um zu bestätigen.
