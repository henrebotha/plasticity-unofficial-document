# Fillet Curve

<Reference shortcut="B" command="Fillet Curve" button="" />

Fillet or chamfer between two Curve segments.

1. Select the Curves where you want to apply fillet or chamfer.
2. Press <kbd>B</kbd> or click the button <ButtonIcon type="button-fillet" /> on the Right of Command Bar.
3. Move the yellow dot to specify the radius of the fillet or chamfer. Depending on the direction of movement, it will either be filleted or chamfered.
4. Set the options in the Command Dialog and Keyboard Shortcuts.
   - **Distance:** Specify the Distance.
   - <kbd>D</kbd> **Fillet Distance:** Specify the fillet distance by moving the cursor.
   - <kbd>C</kbd> **Chamfer Distance:** Specify the Chamfer distance by moving the cursor.
5. Confirm with OK in the Command Dialog or right-click.
