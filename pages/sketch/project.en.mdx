# Project

<Reference shortcut="I" command="Project" button="Right of the command bar" />

Unified command for Project Body Body, Project Curve Body, Project Curve Curve. The command is run based on the selected object.
   - When two Curves are selected: see [Project Curve Curve](./project-curve-curve).
   - When two Solids or Sheets are selected: see [Project Body Body](./project-body-body).
   - Otherwise: see [Project Curve Body](./project-curve-body).
