# Project（投影）

<Reference shortcut="I" command="Project" button="コマンドバー右" />

コマンドを実行したときに選択していたオブジェクトによって挙動が変わります。

- ふたつのカーブを選択していたとき: [**Project Curve Curve**](#project-curve-curve)
- ふたつのソリッドまたはシートを選択していたとき: [**Project Body Body**](#project-body-body)
- その他のとき: [**Project Curve Body**](#project-curve-body)

## Project Curve Curve（ふたつのカーブを投影）[#project-curve-curve]

ふたつのカーブを投影した交線のカーブを作成できます。

1. 投影したいカーブをふたつ選択します。
2. キーボードショートカット<kbd>I</kbd>またはボタンを押します。
3. 投影した交線のカーブが作成されます。

## Project Body Body（ソリッドの交線のカーブを作成）[#project-body-body]

1. ソリッドまたはシートをふたつ選択します。
2. キーボードショートカット<kbd>I</kbd>またはボタンを押します。
3. 交線のカーブが作成されます。

## Project Curve Body（カーブをボディに投影）[#project-curve-body]

カーブをソリッドまたはシートに投影して、カーブを作成できます。

1. 投影したいソリッドまたはシートを選択します。
2. キーボードショートカット<kbd>I</kbd>またはボタンを押します。
3. 投影するカーブを選択します。
4. コマンドダイアログのオプションを設定します。
   - **Method:**
     - **Normal:** 構築平面を基準に Z 軸正方向に投影します。
     - **Vector:** 投影方向を**Direction**で指定します。
          - **Direction:** 構築平面を基準に指定ベクトル方向 (**X**,**Y**,**Z**)に投影します。
   - **Bidirectional:** オンにすると指定した軸の負の方向にも投影されます。
   - **X-Ray:** オンにすると指定した軸の正の方向で、隠れているフェイスには投影されなくなります。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
6. 投影されたカーブが作成されます。
