# Subdivide Curve

<Reference shortcut="Shift + S" command="Subdivide Curve" button="Right of Command Bar" />

Increase the number of Control Vertices(CVs) on the selected Curves.

1. Select one or more Curves.
2. Press <kbd>Shift + S</kbd> or click the button <ButtonIcon type="button-subdivide-curve" /> on the Right of Command Bar.
3. The number of CVs is increased, and they are selected.
