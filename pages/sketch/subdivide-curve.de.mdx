# Subdivide Curve

<Reference shortcut="Umschalt + S" command="Subdivide Curve" button="Rechts von der Befehlsleiste" />

Erhöht die Anzahl der Control Vertices(CVs) auf den ausgewählten Curves.

## Grundlegende Nutzung

1. Wählen Sie einen oder mehrere Curves aus.
2. Drücken Sie <kbd>Umschalt + S</kbd> oder klicken Sie auf die Schaltfläche <ButtonIcon type="button-subdivide-curve" /> rechts von der Befehlsleiste.
3. Die Anzahl der CVs wird erhöht und sie werden ausgewählt.
