# Text

<Reference shortcut="" command="Text" button="" />

Erstellt Curves aus dem eingegebenen Text. Eingebaute Schriftarten und Größen können angegeben werden.

1. Verwenden Sie den **Text** Befehl in der Befehlspalette.
2. Legen Sie die Optionen im Command Dialog fest.
   - <Kst>Befehlsdialog</Kst>
      - **Text box:** Geben Sie den Text ein, den Sie erstellen möchten.
      - **Font:** Wählen Sie die gewünschte Schriftart aus.
      - **Size:** Geben Sie die Schriftgröße an.
3. Bestätigen Sie mit OK im Command Dialog oder Rechtsklick.
4. Die Text Curves werden erstellt.

<Callout emoji="❗">
**Hinweis**<br />
Die Standard-Schriftgröße beträgt 1cm in der Höhe und der Text wird am Ursprung der XY-Ebene positioniert.
</Callout>
