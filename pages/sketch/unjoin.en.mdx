# Unjoin

<Reference shortcut="Alt + J" command="Unjoin" button="" />

Unified command for Unjoin Curves, Unjoin Faces, Unjoin Shells. The command is run based on the selected object.

This command automatically calls the following command depending on the selected object.
    - When Curves are selected: see [Unjoin Curves](./unjoin-curve).
    - When Faces are selected: see [Unjoin Faces](../solid/unjoin-faces) in the Solid section.
    - When Solids or Sheets are selected: see [Unjoin Shells](../solid/unjoin-sheets) in the Solid section.
