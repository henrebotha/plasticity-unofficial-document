# 構築平面に投影

<Reference shortcut="Alt + D" command="Duplicate Curve and Project" button="コマンドバー左" />

カーブまたはエッジを構築平面に投影したカーブを作成できます。

## 基本的な使い方

1. 投影したい構築平面をアクティブにします。
2. 作成する対象として、カーブまたはエッジを選択します
3. キーボードショートカット<kbd>Alt + D</kbd>を押します。
4. 移動ギズモが表示され、投影したたカーブを自由に移動できます。
  - 移動方向を限定することもできます。
  - <Kst>キーボードショートカット</Kst>
    - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
    - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z-plane:** X, Y, Z平面に拘束します。
    - <kbd>G</kbd> **Screen space:** 見た目の平面に限定します。
    - <kbd>F</kbd> **Freestyle:** 最初のクリックで始点、次のクリックで終点を指定します。
    - <kbd>V</kbd> **Pivot:** ギズモの位置をクリックして指定します。
    - <kbd>W</kbd> **Worldspace:** ギズモの座標軸を元に戻します。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
