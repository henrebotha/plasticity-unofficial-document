# Explode

<Reference shortcut="Alt + J" command="Explode" button="" />

Unified command for Explode Curves, Explode Faces, Explode Shells. The command is run based on the selected object.

This command automatically calls the following command depending on the selected object.
    - When Curves are selected: see [Explode Curves](./explode-curve).
    - When Faces are selected: see [Explode Faces](../solid/explode-faces) in the Solid section.
    - When Solids or Sheets are selected: see [Explode Shells](../solid/explode-sheets) in the Solid section.
