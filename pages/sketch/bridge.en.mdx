# Bridge

<Reference shortcut="" command="Bridge" button="" />

Unified command for Bridge Curve, Bridge Vertex, Bridge Edge. The command is run based on the selected object.
     - When two Curves are selected or nothing is selected: see [Bridge Curve](./bridge-curve).
     - When two Vertices are selected: see [Bridge Vertex](./bridge-vertex).
     - When two Edges are selected: see [Bridge Edge](./bridge-edge).
