# Fillet Curve（カーブをフィレットする）

<Reference shortcut="B" command="Fillet Curve" button="コマンドバー右" />

2つのカーブセグメントをフィレットまたは面取りできます。

## 基本的な使い方

1. フィレットまたは面取りをしたいカーブを選択します。
2. <kbd>B</kbd> を押すか、コマンドバー右の <ButtonIcon type="button-fillet" /> をクリックします。
3. 黄色い点を動かすことで、フィレットまたは面取りを作成できます。動かす方向によって、フィレットまたは面取りが作成されます。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
    - <Kst>コマンドダイアログ</Kst>
        - **Distance:** 距離を指定します。
    - <Kst>キーボードショートカット</Kst>
        - <kbd>D</kbd> **Fillet Distance:** カーソルを動かすことで、フィレットの距離を指定します。
        - <kbd>C</kbd> **Chamfer Distance:** カーソルを動かすことで、面取りの距離を指定します。
5. コマンドダイアログの OK をクリックするか、右クリックで確定します。
