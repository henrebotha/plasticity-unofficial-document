## スケッチのコマンド - リージョン・カーブ・頂点・制御点[#sketch-commands]

| コマンド名                                                        | ショートカット       | 機能                                                                                             |
| ----------------------------------------------------------------- | -------------------- | ------------------------------------------------------------------------------------------------ |
| [Fillet](sketch/fillet)                                           | <kbd>B</kbd>         | Fillet Curve, Fillet Vertex, Fillet Shell の統一コマンド                                         |
| [Fillet Curve](sketch/fillet-curve)                               | <kbd>B</kbd>         | 2つのカーブセグメントをフィレットまたは面取りできます                                            |
| [Fillet Vertex](sketch/fillet-vertex)                             | <kbd>B</kbd>         | 接続された頂点間をフィレットまたは面取りできます                                                 |
| [Offset Curve](sketch/offset-curve)                               | <kbd>O</kbd>         | Offset Planar Curve, Offset Region, Offset Face Loop, Offset Edge の統一コマンド                 |
| [Offset Planar Curve](sketch/offset-planar-curve)                 | <kbd>O</kbd>         | 平面カーブを指定した距離でオフセットして新しいカーブを作成できます                               |
| [Offset Region](sketch/offset-region)                             | <kbd>O</kbd>         | 選択したリージョンのアウトラインカーブを距離を指定してオフセットし、新しいリージョンを作成します |
| [Subdivide Curve](sketch/subdivide-curve)                         | <kbd>Shift + S</kbd> | カーブの制御点を増やすことができます                                                             |
| [Extend](sketch/extend)                                           |                      | Extend Curve, Extend Sheet の統一コマンド                                                        |
| [Extend Curve](sketch/extend-curve)                               |                      | カーブを延長できます                                                                             |
| [Rebuild](sketch/rebuild)                                         |                      | Rebuild Curve, Rebuild Face の統一コマンド                                                       |
| [Rebuild Curve](sketch/rebuild-curve)                             |                      | カーブの形状は変えずに制御点の位置を最適化することで、カーブを再構築できます                     |
| [Text](sketch/text)                                               |                      | 入力したテキストでカーブを作成できます                                                           |
| **Project Commands**                                              |                      |                                                                                                  |
| [Project](sketch/project)                                         | <kbd>I</kbd>         | Project Body Body, Project Curve Body, Project Curve Curve の統一コマンド                        |
| [Project Body Body](sketch/project-body-body)                     | <kbd>I</kbd>         | 2つのソリッドまたはシートの交差線からカーブを作成できます                                        |
| [Project Curve Body](sketch/project-curve-body)                   | <kbd>I</kbd>         | カーブをソリッドまたはシートに投影して、カーブを作成できます                                     |
| [Project Curve Curve](sketch/project-curve-curve)                 | <kbd>I</kbd>         | ふたつのカーブを投影した交線のカーブを作成できます                                               |
| [Project Outline](sketch/project-outline)                         | <kbd>Alt + D</kbd>   | ソリッドまたはシートの外形を現在の構築平面に法線の方向に投影し、カーブを生成します               |
| [Alternative Duplicate](sketch/alternative-duplicate)             | <kbd>Alt + D</kbd>   | カーブまたはエッジを構築平面に投影したカーブを作成できます                                       |
| [Create Outline](sketch/create-outline)                           |                      | ソリッドまたはシートの外形を法線方向に沿って現在の構築平面に投影し、3Dのカーブを生成します       |
| [Duplicate Curve and Project](sketch/duplicate-curve-and-project) |                      | Duplicate Curve And Project                                                                      |
| **Join and Split Commands**                                       |                      |                                                                                                  |
| [Join](sketch/join)                                               | <kbd>J</kbd>         | Join Curves, Join Sheets の統一コマンド                                                          |
| [Join Curves](sketch/join-curves)                                 | <kbd>J</kbd>         | 一致した点でカーブをつなぐことができます。                                                       |
| [Bridge](sketch/bridge)                                           |                      | Bridge Curve, Bridge Vertex, Bridge Edge の統一コマンド                                          |
| [Bridge Curve](sketch/bridge-curve)                               |                      | 連続性を考慮してふたつのカーブを繋げることができます                                             |
| [Bridge Edge](sketch/bridge-edge)                                 |                      | 連続性を考慮してふたつのエッジを繋げることができます                                             |
| [Bridge Vertex](sketch/bridge-vertex)                             |                      | 連続性を考慮してふたつの頂点を繋げることができます                                               |
| [Trim](sketch/trim)                                               | <kbd>T</kbd>         | 平面上や3次元的なカーブをセグメントごとにトリムすることができます                                |
| [Cut Curve](sketch/cut-curve)                                     |                      | カーブを、カーブまたはフェイスで切断できます                                                     |
| [Split Segment](sketch/split-segment)                             |                      | カーブをふたつのセグメントに分割することができます                                               |
| [Unjoin](sketch/unjoin)                                           | <kbd>Alt + J</kbd>   | Unjoin Curves, Unjoin Faces, Unjoin Shells の統一コマンド                                        |
| [Unjoin Curves](sketch/unjoin-curve)                              | <kbd>Alt + J</kbd>   | 繋がったカーブを切り離すことができます                                                           |
| **Vertex Commands**                                               |                      |                                                                                                  |
| [Delete Redundant Topology](sketch/delete-redundant-topology)     |                      | カーブの余剰な制御点を削除することができます                                                     |
| [Convert Vertex](sketch/convert-vertex)                           |                      | 頂点を制御点に変換することができます                                                             |
