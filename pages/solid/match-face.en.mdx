# Match Face

<Reference shortcut="" command="Match Face" button="Right of Command Bar" />

Replace the selected Face with another Face.

## Basic Usage

1. Select Faces to match.
2. Click the button <ButtonIcon type="button-match-face" />.
3. Select the surface to replace with.
4. Set the options in the Command Dialog.
    - <Kst>Command Dialog</Kst>
        - **Grow:**
            - When a concave inner Face is offset.
            - **Moving:** Move the whole wall when the offset gets to the outer wall.
            - **Fixed:** Stop at the outer wall.
            - **None:** Keep going by itself, neither affecting or affected by the outer wall.
        - **Side:** 
            - **Front:** When enabled, the Face will be matched to the front Face of the selected surface.
5. Confirm with OK in the Command Dialog or right-click.
