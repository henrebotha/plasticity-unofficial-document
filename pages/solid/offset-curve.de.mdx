# Offset Curve

Einheitlicher Befehl für **Offset Planar Curve**, **Offset Region**, **Offset Face Loop**, **Offset Edge**. Der Befehl wird basierend auf dem ausgewählten Objekt ausgeführt.
   - Wenn Kurven ausgewählt sind: siehe [**Offset Planar Curve**](/sketch/offset-planar-curve) im Abschnitt Skizzierbefehle.
   - Wenn Regionen ausgewählt sind: siehe [**Offset Region**](/sketch/offset-region) im Abschnitt Skizzierbefehle.
   - Wenn Faces ausgewählt sind: siehe [**Offset Face Loop**](offset-face-loop).
   - Wenn Kanten ausgewählt sind: siehe [**Offset Edge**](offset-edge).
