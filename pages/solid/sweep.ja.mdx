# スイープ

<Reference shortcut="Shift + P" command="Sweep" button="コマンドバー右" />

フェイス、リージョンまたはカーブをスイープできます。

1. スイープするリージョン、フェイス、またはカーブを選択します。
2. <kbd>Shift + P</kbd>を押します。
3. スイープのパスとして機能するカーブを選択します。
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - **Twist:**  ねじる角度を度で指定します。
   - **Alignment:**
      - **Parallel:** パスの方向に関係なく平行にスイープします。 
      - **Normal:** パスに沿った角度でスイープします。    
   - <kbd>D</kbd> **Twist** マウスの動きでねじりを指定します。
5. スイープガイドを指定することもできます。
      1. **Select guides**がハイライトされていることを確認します。
      2. スイープのガイドとして機能するカーブを選択します。
         - <kbd>Shift</kbd> + クリックで複数のガイドを選択できます。
         - <kbd>Ctrl</kbd> + クリックで選択を解除できます。
         - ガイドを選択する順序は、スイープの形状に影響します。
      3. コマンドダイアログでオプションを設定します。
         - **Guide method:**
            - **Point:** スイープされたボディの断面は、ガイドとパスと最初に一致していた点が常に接触したままになるように回転およびスケーリングされます。
            - **Chord:**  断面は回転されますが、スケーリングされません。
            - **Curve:** 断面は回転されますが、スケーリングされません。ガイドとパスはプロファイルに接触する必要があります。プロファイルは両方のガイドワイヤーに接触したままになりますが、接触点の1つだけが固定されます。（注: プロファイルは、ふたつのガイドワイヤー間の最大距離を跨ぐのに十分な大きさでなければなりません。）
6. コマンドダイアログのOKをクリックするか、右クリックで確定します。

7. 作成されるオブジェクトは、最初の選択によって異なります。
   - リージョン、フェイスを選択したときは、ソリッドが作成されます。
   - カーブを選択したときは、シートが作成されます。
