# Imprint

<Reference shortcut="Shift + I" command="Imprint" button="" />

Unified command for Imprint Body Body, Imprint Curve Body. The command is run based on the selected object.
     - When Curves or Solid or Sheet are selected or nothing is selected: see [Imprint Curve Body](./imprint-curve-body).
     - When two Solids or Sheets are selected: see [Imprint Body Body](./imprint-body-body).
