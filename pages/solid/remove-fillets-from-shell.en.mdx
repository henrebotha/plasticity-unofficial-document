# Remove Fillets From Shell

Remove fillets from Solid or Sheet with possible restrictions on max radius and convexity.

1. Select the Solid or Sheet you want to remove fillets from.
2. Use the **Remove Fillets From Shell** command in the Command Palette.
3. Set the options in the Command Dialog.
    - **Max Radius:** Specify the maximum radius of fillets to remove.
    - **Convexity:** Specify the convexity of fillets to remove.
        - **Any:** Remove fillets of any convexity.
        - **Convex:** Remove convex fillets.
        - **Concave:** Remove concave fillets.
4. Confirm with OK in the Command Dialog or right-click.
