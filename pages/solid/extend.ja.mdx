# Extend

<Reference shortcut="" command="Extend" button="" />

このコマンドは、選択されていたオブジェクトに応じて以下のコマンドを自動的に呼び出します。
    - カーブが選択されているとき: スケッチコマンドの [**Extend Curve**](../sketch/extend) を参照してください。
    - シートの外側のエッジが選択されているとき: 以下の **Extend Sheet** を参照してください。

## Extend Sheet （シートを延長する）[#extend-sheet]

1. 延長したいシートの外側のエッジを選択します。
2. 外側のエッジが選択されると、デフォルトのコマンドである **Extend Sheet** が自動的に実行されます。
3. 黄色い点を動かして、延長する距離を指定します。
4. コマンドダイアログでオプションを設定します。
    - **Shape:** **Liner** /  **Soft** /  **Reflective**
        - 動画を参照して、それぞれのオプションの違いを確認してください。
    - **Distance:** 延長する距離を指定します。
    - **Modify:** 有効にすると、一枚のシートになります。
5. コマンドダイアログで OK を押すか、右クリックで確定します。
