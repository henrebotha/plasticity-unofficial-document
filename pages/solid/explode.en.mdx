# Explode

<Reference shortcut="Alt + J" command="Explode" button="" />

You can separate joined Curves,Faces, Solids, and Sheets.

Unified command for Explode Curves, Explode Faces, Explode Shells. The command is run based on the selected object.
    - When Curves are selected: see [Explode Curves](../sketch/explode-curve) in the sketch section.
    - When Faces are selected: see [Explode Faces](./explode-faces).
    - When Solids or Sheets are selected: see [Explode Shells](./explode-shells).
