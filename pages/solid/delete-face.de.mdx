# Delete Face

Sie können Faces löschen.

## Grundlegende Nutzung

1. Wählen Sie die Faces aus, die Sie löschen möchten.
2. Drücken Sie <kbd>Umschalt + Entf</kbd>.
3. Die Faces werden gelöscht.
