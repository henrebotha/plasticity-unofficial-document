# Offset Curve

<Reference shortcut="O" command="Offset Curve" button="Right of Command Bar" />

Unified command for **Offset Planar Curve**, **Offset Region**, **Offset Face Loop**, **Offset Edge**. The command is run based on the selected object.
   - When Curves are selected: see [**Offset Planar Curve**](/sketch/offset-planar-curve) in the Sketch Commands section.
   - When Regions are selected: see [**Offset Region**](/sketch/offset-region) in the Sketch Commands section.
   - When Faces are selected: see [**Offset Face Loop**](offset-face-loop).
   - When Edges are selected: see [**Offset Edge**](offset-edge).
