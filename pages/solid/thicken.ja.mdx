# Thicken（シートに厚みをつける）

<Reference shortcut="" command="Thicken" button="Right of Command Bar" />

シートに厚みをつけることができます。
フェイスを選択している場合は、[Thicken Face](./thicken-face)を参照してください。

1. 厚みをつけたいシートを選択します。
2. コマンドバーの右側にある <ButtonIcon type="button-thicken" /> ボタンをクリックします。
3. コマンドダイアログとキーボードショートカットでオプションを設定します。
    - **Thickness:** 両側のそれぞれの厚みを指定します。
    - <kbd>Tab</kbd> **Lock Distances** 両側の厚みを等しくします。
    - <kbd>D</kbd> **Thickness** カーソルを動かして厚みを指定します。
4. コマンドダイアログの OK または右クリックで確定します。
