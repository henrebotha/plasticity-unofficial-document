# Hollow

<Reference shortcut="" command="Hollow" button="Rechts von der Befehlsleiste" />

Wenn eine Fläche ausgewählt ist: Höhlt die ausgewählte Fläche aus und schafft eine offene Leere im Solid. <br />
Wenn ein Solid ausgewählt ist: Verwandelt den ausgewählten Solid in eine geschlossene hohle Form.

## Grundlegende Nutzung

### Einen offenen hohlen Solid erstellen

1. Wählen Sie die Flächen des Solids aus, die Sie aushöhlen möchten.
2. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-hollow" />
3. Bewegen Sie den gelben Punkt, um die Dicke der Wand anzugeben.
4. Legen Sie die Optionen im Befehlsdialog fest:
  - <Kst>Befehlsdialog</Kst>
    - **Dicke:** Geben Sie die Dicke der Wand an.
    - **Lokal:** Deaktivieren Sie dies, wenn Sie einen Solid aushöhlen.
  - <Kst>Tastenkombinationen</Kst>
    - <kbd>D</kbd> Geben Sie die Dicke durch Bewegen des Cursors an.
5. Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.
6. Ein geöffneter hohler Solid wird erstellt.

### Einen geschlossenen hohlen Solid erstellen

1. Wählen Sie einen Solid aus, den Sie aushöhlen möchten.
2. Verwenden Sie den **Hohl**-Befehl in der Befehlspalette.
3. Bewegen Sie den gelben Punkt, um die Dicke der Wand anzugeben.
  - <Kst>Befehlsdialog</Kst>
    - **Dicke:** Geben Sie die Dicke der Wand an.
    - **Lokal:** Deaktivieren Sie dies, wenn Sie einen Solid aushöhlen.
  - <Kst>Tastenkombinationen</Kst>
    - <kbd>D</kbd> Geben Sie die Dicke durch Bewegen des Cursors an.
5. Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.
6. Ein geschlossener hohler Solid wird erstellt.
