# Isoparam

<Reference shortcut="Strg + R" command="Isoparam" button="" />

Kanten auf einer Face eines Solid oder Sheet in der U- oder V-Richtung erstellen. Mehrere Kanten können gleichzeitig erstellt werden.

## Grundlegende Nutzung

1. Drücken Sie <kbd>Strg + R</kbd>.
2. Platzieren Sie den Mauszeiger an der Position, an der Sie die Isoparams erstellen möchten.
3. Legen Sie die Optionen in den Tastenkombinationen fest.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd> **U/V direction:** Wechseln Sie die Richtung des zu erstellenden Isoparam.
      - <kbd>Umschalt</kbd> + Mausrad hoch **Add cut:** Erhöht die Anzahl der Isoparams.
      - <kbd>Umschalt</kbd> + Mausrad runter **Subtract cut:** Reduziert die Anzahl der Isoparams.
4. Bestätigen Sie mit einem Rechtsklick.
