# Constrained Surface

<Reference shortcut="" command="Constrained Surface" button="" />

Generate a surface that passes through a set of points by clicking on them.

1. Use the **Constrained Surface** command in the Command Palette.
2. Click on the points you want to pass through.
3. Set the options in the Command Dialog and Keyboard Shortcuts.
    - **Tolerance:** Specify the tolerance of the surface.
    - **Angular tolerance:** Specify the angular tolerance of the surface.
    - **Optimize:**
        - **Performance:** The surface is generated quickly.
        - **Smoothness:** The surface is generated smoothly.
    - <kbd>Cmd-Z</kbd>: Undo the last point.
4. Confirm with OK in the Command Dialog or right-click.
