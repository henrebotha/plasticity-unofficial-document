# Refillet Face

<Reference shortcut="" command="Refillet Face" button="" />

Redo fillet without selecting individual fillet Faces from Solid or Sheet. Distance is adjustable.

1. Select all surfaces, including the fillet surfaces.
2. Use the **Refillet Face** command in the Command Palette.
3. Move the yellow dot to specify the radius of the fillet.
4. Set the options by Keyboard Shortcuts.
    - <kbd>D</kbd> **Distance:** Specify the distance by moving the cursor.
5. Confirm with a right-click.
