# Alternative Duplicate（代替複製する）

<Reference shortcut="Alt + D" command="Alternative Duplicate" button="コマンドバー左" />

選択したフェイスを元にソリッドを作成することができます。

1. ソリッドを作成したい元になるフェイスを選択します。
2. コマンドパレットで**Alternative Duplicate**を実行します。
3. ソリッドが作成されます。表示された移動ギズモを使って移動します。
4. コマンドダイアログやキーボードショートカットでオプションを設定します。
   - **XYZ:** X、Y、Z軸方向の移動量を指定します。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
   - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z-plane:** X, Y, Z平面に拘束します。
   - <kbd>G</kbd> **Screen space:** 見た目の平面に限定します。
   - <kbd>F</kbd> **Freestyle:** 始点と終点をクリックして、移動の方向と距離を指定します。
   - <kbd>V</kbd> **Pivot:** クリックしてギズモの位置を指定します。
   - <kbd>W</kbd> **Worldspace:** ギズモの座標軸をワールド座標にします。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
