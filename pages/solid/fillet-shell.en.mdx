# Fillet Shell

<Reference shortcut="B" command="Fillet Shell" button="" />

Fillet or chamfer the Edges which can be conic or G2 continuous. Variable distance, limit range are adjustable.

## Basic Usage

1. Select the Edges to fillet or chamfer.
2. When one or more Edges are selected, the default command, **Fillet**, is automatically executed.
3. Alternatively, press the shortcut key <kbd>B</kbd> to excute.
4. Move the yellow dot to specify the distance of the fillet or chamfer. 
    - Positive movement:  results in a fillet.
    - Negative movement: results in a chamfer.
    ![Fillet](/img/fillet-solid-0.gif)
5. Optionally, to select more Edges, <kbd>Ctrl</kbd> + click on the Edge. To deselect, <kbd>Ctrl</kbd> + click again.
6. Set the options in the Command Dialog and Keyboard Shortcuts. See [**Fillet options**](#fillet-options) and [**Chamfer options**](#chamfer-options) below.
7. Confirm with OK in the Command Dialog or right-click.

## Fillet options

### Command Dialog

**Distance:** 
    - To fillet an Edge, enter a positive value. To chamfer an Edge, enter a negative value
<Callout type="info" emoji="üí°">
    **Tip**
    To copy a fillet radius, click on the source fillet, but only if the two Distance values are the same and the shape is a Conic.
![](/img/fillet-solid-copyfillet.gif)
</Callout>

**Shape:**
    - **Conic:** The Conic fillet option is known for producing a more natural and smooth transition compared to traditional round fillets. This makes it especially useful in applications where aerodynamics or aesthetics are crucial.
    - **Chordal:** The hallmark of a Chordal fillet is that it maintains a constant length for the chord of the fillet, regardless of the angles being filleted. This uniformity can be particularly beneficial when aiming for visual consistency across different parts of a model.
    - **G2:** The G2 option generates curves that are smoothly continuous with the surfaces at both ends of the fillet. This setting is optimal for intricate designs and complex surface handling that require an extremely smooth and natural shape transition. See [Surface Continuity](../cad-essentials/continuity-curve-and-surface).
    <VideoPlayer fileName="fillet-shell-shape"/>

**Y-Blend:** 
    - The ‚ÄúY-blend‚Äù is a special kind of blend or fillet that occurs when three surfaces meet at a common point, forming a ‚ÄúY‚Äù shape. Enabling **Attempt**, you can create better topology  in Y-blend and avoid undesired fillet results.
    ![](/img/fillet-solid-yblend.gif)

**Selection:** 
    - Disable **Add tangent Edges** to limit fillets to selected Edges only.
    - Shortcut <kbd>T</kbd> is acceptable.
    ![](/img/fillet-sold-selection.gif)

### Shortcuts

<kbd>A</kbd> **: Chamfer angle** 
    - Specify fillet angle by moving the cursor.
<Callout type="info" emoji="üí°">
    Also, changing the two values of distance changes the angle.
    ![](/img/fillet-solid-two-value.gif)
</Callout>

<kbd>D</kbd> **: Fillet distance** 
    - Specify Fillet distance by moving the cursor.<br />

<kbd>C</kbd> **: Chamfer distance** 
    - Specify Chamfer distance by moving the cursor.<br />

<kbd>V</kbd> **: Add variable point** <br />
    - Add a variable point to the fillet.<br />
    ![](/img/fillet-solid-variable.gif)

<kbd>L</kbd> **: Add limit (stop) point** <br />
    - Specify the range to fillet. Make sure that [**Object Snaps**](../plasticity-essentials/grid-and-snap#object-snap) are enabled, Click on the center line of fillet to add a limit point. To reverse the direction, click on the limit point again. Up to two limit points can be set.
    ![](/img/fillet-solid-limit.gif)

## Chamfer options
- **Shape:**
    - **Offset:** Default chamfer.
    - **Apex:** Reduce the displacement of the border of the chamfer surface.
- Others are the same as fillet options.
