# Hollow

<Reference shortcut="" command="Hollow" button="Right of Command Bar" />

When a Face is selected: Hollows out the selected Face, creating an open void in the Solid. <br />
When a Solid is selected: Turns the selected Solid into a closed hollow form.

## Create an Open Hollow Solid

1. Select a Faces of the Solid you want to hollow out.
2. Click the button <ButtonIcon type="button-hollow" />
3. Move the yellow dot to specify the thickness of the wall.
4. Set the options in the Command Dialog
  - **Thickness:** Specify the thickness of the wall.
  - **Local:** Disable when hollowing out a Solid.
  - <kbd>D</kbd> Specify the thickness by moving the cursor.
5. Confirm with OK in the Command Dialog or right-click.
6. An opened hollow Solid is created.

![hollow](/img/hollow.gif)

## Create a Closed Hollow Solid

1. Select a Solid you want to hollow out.
2. Use **Hollow** command in the Command Palette.
3. Move the yellow dot to specify the thickness of the wall.
  - **Thickness:** Specify the thickness of the wall.
  - **Local:** Disable when hollowing out a Solid.
  - <kbd>D</kbd> Specify the thickness by moving the cursor.
5. Confirm with OK in the Command Dialog or right-click.
6. A closed hollow Solid is created.
