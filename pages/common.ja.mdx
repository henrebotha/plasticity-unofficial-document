## スケッチとソリッド 共通のコマンド
| コマンド名                                          | ショートカット              | 機能                                   |
| --------------------------------------------------- | --------------------------- | -------------------------------------- |
| [Move](common/move)                                 | <kbd>G</kbd>                | オブジェクトを移動する                 |
| [Rotate](common/rotate)                             | <kbd>R</kbd>                | オブジェクトを回転する                 |
| [Scale](common/scale)                               | <kbd>S</kbd>                | オブジェクトを拡大縮小する             |
| [Mirror](common/mirror)                             | <kbd>Alt + X</kbd>          | オブジェクトをミラーする               |
| [Duplicate](common/duplicate)                       | <kbd>Shift + D</kbd>        | オブジェクトを複製する                 |
| [Place](common/place)                               | <kbd>Ctrl + D</kbd>         | オブジェクトを指定軸を基準に複製する   |
| [Copy with Placement](common/copy-with-placement)   | <kbd>Ctrl + Shift + C</kbd> | 参照点を用いてオブジェクトをコピーする |
| [Paste with Placement](common/paste-with-placement) | <kbd>Ctrl + Shift + V</kbd> | 参照点を用いてオブジェクトを貼り付ける |
| [Rectangular Array](common/rectangular-array)       |                             | オブジェクトを長方形配列で複製する     |
| [Radial Array](common/radial-array)                 |                             | オブジェクトを円形状配列で複製する     |
| [Curve Array](common/curve-array)                   |                             | オブジェクトをカーブに沿って複製する   |
| [Set Material](common/set-material)                 | <kbd>M</kbd>                | マテリアルを設定する                   |
| [Remove Material](common/remove-material)           | <kbd>Alt + M</kbd>          | マテリアルを削除する                   |
