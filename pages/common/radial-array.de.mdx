# Radial Array

<Reference shortcut="" command="Radial Array" button="Rechts von der Befehlsleiste" />

Duplizieren Sie Objekte in einem kreisförmigen Muster, indem Sie den Winkel und die Anzahl angeben. Radiale Duplizierung ist ebenfalls möglich.

## Grundlegende Nutzung

1. Wählen Sie das Objekt zum Duplizieren aus.
2. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-radial-array" /> rechts von der Befehlsleiste.
3. Klicken Sie auf den Punkt, den Sie als Zentrum festlegen möchten.
4. Die Duplikationsvorschau wird angezeigt.
5. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - <Kst>Befehlsdialog</Kst>
      - **Degrees:** Geben Sie den Winkel für die Duplizierung an.
      - **Number:** Geben Sie die Anzahl der Duplizierungen an.
      - **Repeat:** Geben Sie die Anzahl der Wiederholungen in einem radialen Muster an.
      - **Repeat distance:** Geben Sie den Abstand zwischen jeder Wiederholung in einem radialen Muster an.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Umschalt</kbd> + Mausrad hoch **Add copy:** Erhöht die Anzahl der Duplizierungen.
      - <kbd>Umschalt</kbd> + Mausrad runter **Subtract copy:** Verringert die Anzahl der Duplizierungen.   
6. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
