# Scale（拡大縮小）

<Reference shortcut="S" command="Scale" button="コマンドバー左" />

オブジェクトをScale Gizmoを使用してスケーリングできます。単一軸の制約や[フリースタイルの拡大縮小](#freestyle-scale)のオプションが利用可能です。また、ピボットも調整できます。

1. 拡大縮小したいオブジェクトを選択します。
2. キーボードショートカット <kbd>S</kbd> を押します。
3. 表示された拡大縮小ギズモを使用して大きさを変更します。 
4. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - **XYZ:** X、Y、Z軸方向の拡大率を指定します。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
   - <kbd>Shift + X</kbd>/ <kbd>Shift + Y</kbd>/<kbd>Shift + Z</kbd> **X, Y, Z-plane:** X, Y, Z平面に拘束します。
   - <kbd>S</kbd> **Uniform:** X、Y、Z方向に均等に拡大縮小します。
   - <kbd>F</kbd> **Freestyle:** 始点と終点をクリックして拡大縮小の軸を指定し、希望の拡大率でクリックします。
   - <kbd>V</kbd> **Pivot:** クリックしてギズモの位置を指定します。
   - <kbd>W</kbd> **Worldspace:** ギズモの座標軸をワールド座標にします。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。

## Freestyle Scale[#freestyle-scale]

このコマンドは **Scale** コマンドの<kbd>F</kbd>オプションと同じです。

1. 拡大縮小したいオブジェクトを選択します。
2. コマンドパレットで **Freestyle Scale** を実行します。
3. 始点と終点をクリックして拡大縮小の軸を指定します。
4. カーソルを動かして拡大縮小し、希望の拡大率でクリックします。
5. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - オプションは **Scale** コマンドと同じです。
6. コマンドダイアログのOKをクリックするか、右クリックで確定します。
