# Copy with Placement

<Reference shortcut="Ctrl + Shift + C" command="Copy With Placement" button="" />

Copy objects with a reference point. Use it alongside the [**Paste with Placement**](paste-with-placement) command for similar functionality to the [**Place**](place) command. This is particularly useful for placing objects in different windows.

## Basic Usage

1. **Copy with Placement** Command
    1. Select the objects you wish to copy.
    2. Press <kbd>Ctrl + Shift + C</kbd>.
    3. Click the reference point on the object you're copying.
2. To paste in a different window, switch to that target window.
3. **Paste with Placement** Command
    1. Press <kbd>Ctrl + Shift + V</kbd>.
    2. Click the destination reference point in the target window.
        - <Kst>Keyboard Shortcuts</Kst> 
            - <kbd>F</kbd> **Flip:** Allows flipping the duplication axis.
            - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **x/y/z-up:** Sets the duplication axis to the X/Y/Z-axis.
            - <kbd>S</kbd> **Scale:** Allows scaling.
            - <kbd>A</kbd> **Angle:** Allows rotation.
            - <kbd>D</kbd> **Offset:** Allows consecutive duplication.
            - <kbd>Shift + Q</kbd> **Slice:** Creates a Solid at the intersections with the Boolean target and preserves the remaining Solids.
            - <kbd>Q</kbd> **Union:** Joins the Boolean target and tool together.
            - <kbd>W</kbd> **Difference:** Removes the tool from the Boolean target.
            - <kbd>Shift + E</kbd> **Intersect:** Creates a Solid at the intersection of the Boolean target and tool.
            - <kbd>B</kbd> **New Body:** Creates a new Solid without performing Boolean operations.
    3. Right-click to confirm.
    4. To copy additional objects, repeat steps 3-2 and 3-3.
    5. Press <kbd>Esc</kbd> to exit the command.
