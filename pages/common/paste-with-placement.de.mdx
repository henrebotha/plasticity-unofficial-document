# Paste with Placement

<Reference shortcut="Strg + Umschalt + V" command="Paste With Placement" button="" />

Fügen Sie Objekte mit einem Referenzpunkt ein. Verwenden Sie es zusammen mit dem [**Copy with Placement**](./copy-with-placement) Befehl, um eine ähnliche Funktionalität wie beim [**Place**](./place) Befehl zu erreichen. Dies ist besonders nützlich, um Objekte in verschiedenen Fenstern zu platzieren.

## Grundlegende Nutzung

1. **Copy with Placement** Befehl
    1. Wählen Sie die Objekte aus, die Sie kopieren möchten.
    2. Drücken Sie <kbd>Strg + Umschalt + C</kbd>.
    3. Klicken Sie auf den Referenzpunkt des zu kopierenden Objekts.
2. Um in einem anderen Fenster einzufügen, wechseln Sie zu diesem Ziel-Fenster.
3. **Paste with Placement** Befehl
    1. Drücken Sie <kbd>Strg + Umschalt + V</kbd>.
    2. Klicken Sie auf den Ziel-Referenzpunkt im Ziel-Fenster.
        - <Kst>Tastenkombinationen</Kst>
            - <kbd>F</kbd> **Flip:** Erlaubt das Spiegeln der Duplikationsachse.
            - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **x/y/z-up:** Legt die Duplikationsachse auf die X/Y/Z-Achse fest.
            - <kbd>S</kbd> **Scale:** Erlaubt das Skalieren.
            - <kbd>A</kbd> **Angle:** Erlaubt das Drehen.
            - <kbd>D</kbd> **Offset:** Erlaubt aufeinanderfolgende Duplikation.
            - <kbd>Umschalt + Q</kbd> **Slice:** Erstellt einen Solid an den Schnittstellen mit dem Boolean-Ziel und bewahrt die verbleibenden Solids.
            - <kbd>Q</kbd> **Union:** Verbindet das Boolean-Ziel und das Werkzeug.
            - <kbd>W</kbd> **Difference:** Entfernt das Werkzeug vom Boolean-Ziel.
            - <kbd>Umschalt + E</kbd> **Intersect:** Erstellt einen Solid an der Schnittstelle des Boolean-Ziels und des Werkzeugs.
            - <kbd>B</kbd> **New Body:** Erstellt einen neuen Solid ohne Boolean-Operationen auszuführen.
    3. Klicken Sie mit der rechten Maustaste, um zu bestätigen.
    4. Um zusätzliche Objekte zu kopieren, wiederholen Sie die Schritte 3-2 und 3-3.
    5. Drücken Sie <kbd>Esc</kbd>, um den Befehl zu beenden.
