# Dimension

<Reference shortcut="Shift + =" command="Dimension" button="" />

Maße ändern, wie zum Beispiel die Größe einer rechteckigen Kurve oder den Durchmesser eines Zylinders.

## Grundlegende Nutzung

1. Wählen Sie das Objekt aus, dessen Maße Sie ändern möchten.
    | Maßänderungsziel                                  | Auswahlmodus |
    | ------------------------------------------------- | ------------ |
    | Länge von zwei Seiten eines Kurvenrechtecks       | Edge         |
    | Durchmesser eines Kreises oder Bogens einer Kurve | Edge         |
    | Durchmesser eines Zylinders                       | Face         |
    | Durchmesser einer Kugel                           | Solid        |
    | Längen der drei Seiten eines Würfels              | Solid        |
    | Größe der Rundung eines Festkörpers               | Face         |
    | Abstand zwischen den Ebenen eines Festkörpers     | Face         |
2. Drücken Sie <kbd>Umschalt + =</kbd> oder verwenden Sie den **Dimension**befehl in der Command Palette.
3. Der Maßwert wird angezeigt.
4. Drücken Sie <kbd>Tab</kbd>, um in den Maßeingabemodus zu wechseln.
5. Geben Sie den Maßwert ein.
6. Wenn es mehrere Maße gibt, drücken Sie <kbd>Tab</kbd>, um zu wechseln und die Maße einzugeben.
7. Drücken Sie <kbd>Eingabe</kbd>, um zu bestätigen.
