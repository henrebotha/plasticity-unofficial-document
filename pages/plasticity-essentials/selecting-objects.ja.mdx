# オブジェクトを選択する

## マウスで選択する

- クリック: 選択できます。
- 右から左に向かってドラッグして選択： 緑色の選択範囲に触れたオブジェクトが選択されます。
- 左から右に向かってドラッグして選択： 紫色の選択範囲に完全に含まれるオブジェクトのみが選択されます。

![Mouse Drag selection](/img/mouse-drag-selection.gif)

## キーボードショートカットで選択する

| Command Name      | Shortcut                    | Note                                                               |
| ----------------- | --------------------------- | ------------------------------------------------------------------ |
| Select all        | <kbd>A</kbd>                | すべてのオブジェクトを選択する                                     |
| Select all Curves | <kbd>Ctrl + A</kbd>         | すべてのカーブを選択する                                           |
| Deselect all      | <kbd>Escape</kbd>           | すべての選択を解除する                                             |
| Invert Selection  | <kbd>Alt + A</kbd>          | 選択を反転する                                                     |
| Select Adjacent   | <kbd>Ctrl + Shift + =</kbd> | 現在選択されているオブジェクトに隣接するオブジェクトを選択します。 |

| Note                                           | Shortcut                      |
| ---------------------------------------------- | ----------------------------- |
| エッジループを選択する                         | <kbd>Alt</kbd> + click        |
| エッジリングを選択する                         | <kbd>Alt + Ctrl</kbd> + click |
| 同一平面のすべてのカーブとリージョンを選択する | <kbd>Alt</kbd> + click        |
| すべての平行なフェイスを選択する               | <kbd>Alt + Ctrl</kbd> + click |

### 選択しているオブジェクトから選択する

| Command Name                | Shortcut            | Note                                                               |
| --------------------------- | ------------------- | ------------------------------------------------------------------ |
| selection-convert-to-edges  | <kbd>Ctrl + 2</kbd> | 選択しているソリッド、シート、フェイスの、エッジのみを選択します   |
| selection-convert-to-faces  | <kbd>Ctrl + 3</kbd> | 選択しているソリッド、シートの、フェイスのみを選択します           |
| selection-convert-to-solids | <kbd>Ctrl + 4</kbd> | 選択しているフェイス、エッジの、ソリッドまたはシートを選択します。 |
