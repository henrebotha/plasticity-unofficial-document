# P-Menu

The P-Menu can be opened by clicking the P at the top left.

![P-Menu](/img/file-menu.png)

| Command Name          | Shortcut                    | Function                                                                                                                                                                                         |
| --------------------- | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| New                   | <kbd>Ctrl + N</kbd>         | Clears the scene and opens a new document.                                                                                                                                                       |
| New Window            |                             | Launches Plasticity and opens a new window. The accent color of the new window is randomly changed to differentiate it from the primary window. Copy and paste between windows is also possible. |
| Open                  | <kbd>Ctrl + O</kbd>         | Opens a Plasticity file.                                                                                                                                                                         |
| Import/Append         | <kbd>Ctrl + Shift + O</kbd> | Imports files. Refer to [here](import-export#file-import) for importable formats.                                                                                                              |
| Save                  | <kbd>Ctrl + S</kbd>         | Saves the current document.                                                                                                                                                                      |
| Save as               | <kbd>Ctrl + Shift + S</kbd> | Saves the document with a new name.                                                                                                                                                              |
| Export                | <kbd>Ctrl + Shift + E</kbd> | Exports files. Refer to [here](import-export#file-export) for exportable formats.                                                                                                              |
| Quick save version    |                             | Saves a copy of the file with a timestamp appended to the file name.                                                                                                                             |
| Save as startup scene |                             | Saves the current document as the startup scene, which will automatically open on the next launch.                                                                                               |
| Preferences           | <kbd>Ctrl + , (Comma)</kbd> | Refer to [settings](preferences).                                                                                                                                                              |
| Manage licenses       |                             | Manages license tokens and devices.                                                                                                                                                              |
| Plasticity Version    |                             | Shows the installed version of Plasticity.                                                                                                                                                       |
| Quit                  | <kbd>Ctrl + Q</kbd>         | Exits Plasticity.                                                                                                                                                                                |
