# P-Menu

To open the P-Menu, click on the P in the upper left corner of the window.

![P-Menu](/img/file-menu.png)

| Command Name              | Shortcut                    | Note                                                                                                        |
| ------------------------- | --------------------------- | ----------------------------------------------------------------------------------------------------------- |
| New                       | <kbd>Ctrl + N</kbd>         | Clear the scene, then create a new Plasticity document.                                                     |
| Open                      | <kbd>Ctrl + O</kbd>         | Open a Plasticity document.                                                                                 |
| Import/Append             | <kbd>Ctrl + Shift + O</kbd> | Import a document. See [File import](#file-import) below.                                                   |
| Save                      | <kbd>Ctrl + S</kbd>         | Save a Plasticity document.                                                                                 |
| Export/Save as            | <kbd>Ctrl + Shift + S</kbd> | Save as a new Plasticity document. Alternatively, export a document. See [File export](#file-export) below. |
| Quick save version        |                             | Save a copy of the document with the same filename, appended with a timestamp.                              |
| Save as startup scene     |                             | Save the current document as the startup scene. The file will automatically open upon the next startup.     |
| Preferences               | <kbd>Ctrl + , (Comma)</kbd> | See [Preferences](./preferences) page.                                                                      |
| Manage licenses           |                             | Manage the license token and devices.                                                                       |
| Plasticity Version: 0.0.0 |                             | Installed Plasticity version.                                                                               |
| Quit                      | <kbd>Ctrl + Q</kbd>         |                                                                                                             |


## File Import [#file-import]
- **Indie and Studio** 
    - Plasticity file (*.plasticity)
    - STEP file (*.stp;*.step)
    - ParaSolid file (*.x_t;*.x_b)
    - Wavefront OBJ file (*.obj)
    - Image files (*.png;*.jpg;*.jpeg)
- **Studio only**
    - IGES files (*.igs;*.iges) 
    - SAT files (*sat)
    - Rhino files (*.3dm) 

## File Export [#file-export]
- **Indie and Studio** 
    - Plasticity file (*.plasticity)
    - STEP file (*.stp;*.step)
    - ParaSolid file (*.x_t;*.x_b)
    - [Wavefront OBJ file](./export-obj) (*.obj)
    - STL file (*.stl)
- **Studio only**
    - IGES file (*.igs;*.iges)
    - SAT file (*sat)
