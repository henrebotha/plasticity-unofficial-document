# 3Dビューポートの操作

## マウスによる操作

| 操作                                       | マウス                                                 |
| ------------------------------------------ | ------------------------------------------------------ |
| ビューを回転する                           | マウスの中ボタン（スクロールホイール）でドラッグします |
| ビューを平行移動する                       | マウスの右ボタンでドラッグします                       |
| ビューを拡大縮小する                       | スクロールホイールを回転します                         |
| 回転中に正投影でスナップする:              | <kbd>Alt</kbd> + マウスの中ボタンでドラッグします      |
| カーソルの位置が中心になるよう画面移動する | <kbd>Alt</kbd> + マウスの中ボタンでクリックします      |

他のアプリケーションと同様のマウスにすることもできます。詳しくは[環境設定](preferences)を参照してください。

## キーボードショートカットによる操作

### ビューのアングルを変える

テンキーでビューの方向を調整できます。

| ビューの方向                 | ショートカット          |
| ---------------------------- | ----------------------- |
| 前から                       | <kbd>Num.1</kbd>        |
| 右から                       | <kbd>Num.3</kbd>        |
| 上から                       | <kbd>Num.7</kbd>        |
| 後ろから                     | <kbd>Ctrl + Num.1</kbd> |
| 左から                       | <kbd>Ctrl + Num.3</kbd> |
| 下から                       | <kbd>Ctrl + Num.7</kbd> |
|                              |                         |
| 透視投影と正投影を切り替える | <kbd>Num.5</kbd>        |
