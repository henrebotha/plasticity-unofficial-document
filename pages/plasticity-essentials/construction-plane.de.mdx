# Konstruktionsebenen

Kurven werden auf der aktiven Konstruktionsebene erstellt. 
Konstruktionsebenen können die XY-, YZ- oder XZ-Ebenen sein, und Sie können auch neue erstellen.

### Verwendung der XY-, YZ- und XZ-Ebenen

- <kbd>Ziffernblock 7</kbd>: Verwenden Sie die XY-Ebene. Dies gibt Ihnen eine Draufsicht.
- <kbd>Ziffernblock 3</kbd>: Verwenden Sie die YZ-Ebene. Dies gibt Ihnen eine Ansicht von rechts.
- <kbd>Ziffernblock 1</kbd>: Verwenden Sie die XZ-Ebene. Dies gibt Ihnen eine Frontansicht.

### Erstellung einer Konstruktionsebene ausgerichtet mit einer Oberfläche

Sie können eine Konstruktionsebene erstellen, die mit dem Gesicht eines Festkörpers oder Blattes ausgerichtet ist.

1. Wählen Sie das Gesicht oder die Region aus, das/die Sie als Konstruktionsebene verwenden möchten.
2. Drücken Sie <kbd>Leertaste</kbd>.
3. Die Konstruktionsebene wird erstellt.
4. Ihre Ansicht wird mit der erstellten Konstruktionsebene ausgerichtet.

- Wenn Sie Ihre Ansicht nicht ändern möchten, drücken Sie <kbd>Shift + Leertaste</kbd> anstelle von <kbd>Leertaste</kbd>.

### Erstellung einer Konstruktionsebene aus mehreren Punkten

Sie können eine Konstruktionsebene basierend auf drei oder mehr Punkten erstellen.

1. Wählen Sie mindestens drei Punkte aus, die Sie als Konstruktionsebene verwenden möchten.
2. Drücken Sie <kbd>Leertaste</kbd>.
3. Wenn Sie drei Punkte ausgewählt haben, wird eine Konstruktionsebene an der genauen Position erstellt. Wenn Sie vier oder mehr Punkte ausgewählt haben, wird die Ebene an ihrer durchschnittlichen Position erstellt.
4. Ihre Ansicht wird mit der erstellten Konstruktionsebene ausgerichtet.

- Wenn Sie Ihre Ansicht nicht ändern möchten, drücken Sie <kbd>Shift + Leertaste</kbd> anstelle von <kbd>Leertaste</kbd>.
- Sie können auch eine Region einer Skizze auswählen, um eine Konstruktionsebene zu erstellen, die damit ausgerichtet ist.

### Erstellung einer Konstruktionsebene ausgerichtet mit der aktuellen Ansicht

Sie können eine Konstruktionsebene erstellen, die mit Ihrer aktuellen Ansicht im Arbeitsbereich ausgerichtet ist.

#### Erstellung einer Konstruktionsebene durch den Weltursprung

1. Richten Sie Ihre Ansicht auf die gewünschte Sichtlinie für die Konstruktionsebene aus.
2. Drücken Sie <kbd>Strg + Leertaste</kbd>.
3. Die Konstruktionsebene wird erstellt.

#### Erstellung einer Konstruktionsebene durch ausgewählten Punkt

1. Richten Sie Ihre Ansicht auf die gewünschte Sichtlinie für die Konstruktionsebene aus.
2. Drücken Sie <kbd>Strg + Shift + Leertaste</kbd>.
3. Klicken Sie auf den Ursprung der Konstruktionsebene, die Sie erstellen möchten.
4. Die Konstruktionsebene wird erstellt.

### Speichern und Verwenden von Konstruktionsebenen

Sie können Ihre erstellten Konstruktionsebenen für eine spätere Verwendung speichern.

#### Speichern von Konstruktionsebenen

1. Klicken Sie oben auf dem Bildschirm auf **Temporary construction plane**.
2. **Custom plane 0** wird auf der rechten Seite des Bildschirms hinzugefügt.

#### Verwenden von gespeicherten Konstruktionsebenen

1. Klicken Sie auf der rechten Seite des Bildschirms auf **Custom plane 0**.
2. Die gespeicherte Konstruktionsebene wird aktiviert.

- Ein Doppelklick auf **Custom plane 0** aktiviert die gespeicherte Konstruktionsebene und richtet Ihre Ansicht darauf aus.
