# Allgemeine Funktionen

| Befehlsname               | Tastenkombination                                         | Anmerkung                                                                                                  |
| ------------------------- | --------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------- |
| Copy                      | <kbd>Strg + C</kbd>                                       |                                                                                                            |
| Paste                     | <kbd>Strg + V</kbd>                                       |                                                                                                            |
| Undo                      | <kbd>Strg + Z</kbd>                                       |                                                                                                            |
| Redo                      | <kbd>Strg + Umschalt + Z</kbd>                            |                                                                                                            |
| Delete                    | <kbd>Umschalt + Rücktaste</kbd>                           | Ausgewählte Objekte löschen. Das Löschen einer Gruppe entfernt die Gruppe und ihre Kinder.                 |
| Delete Control Point      |                                                           | Ausgewählte Kontrollpunkte löschen.                                                                        |
| Delete Edge               |                                                           | Ausgewählte Kanten löschen.                                                                                |
| Delete Face               | <kbd>Umschalt + X</kbd> oder <kbd>Umschalt + Entf</kbd> | Ausgewählte Flächen löschen.                                                                               |
| Delete Group              |                                                           | Entfernt die Gruppe und ihre Kinder.                                                                       |
| Dissolve                  | <kbd>Rücktaste</kbd>                                      | Ausgewählte Objekte löschen. Das Auflösen einer Gruppe entfernt die Gruppe, behält aber ihre Elemente bei. |
| Dissolve Face             |                                                           | Ausgewählte überflüssige Flächen löschen.                                                                  |
| Edit: repeat last Command | <kbd>Umschalt + R</kbd>                                   | Den letzten Befehl wiederholen.                                                                            |
|                           |                                                           |                                                                                                            |
