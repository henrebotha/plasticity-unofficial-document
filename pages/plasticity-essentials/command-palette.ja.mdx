# コマンドパレット
- ボタンやキーボードショートカットのないコマンドはコマンドパレットから実行できます。作業領域で<kbd>F</kbd>キーを押すとコマンドパレットが表示されます。
- コマンド名を入力して実行します。さらに、任意のコマンドにキーボードショートカットを割り当てたり、お気に入りのコマンドをピン留めすることもできます。

## コマンドの実行
1. <kbd>F</kbd>キーを押します。
2. コマンドパレットが表示されます。
3. コマンド入力欄にコマンド名を入力します。
4. 入力した単語を含むコマンドが表示されます。入力ごとに候補が絞られていきます。
5. 上下の矢印キーと<kbd>Enter</kbd>で目的のコマンドを選択します。
6. コマンドが実行されます。

![Command Palette](./img/command-palette.gif)

## キーボードショートカットの割り当て
1. <kbd>F</kbd>キーを押します。
2. コマンドパレットが表示されます。
3. コマンド入力欄にコマンド名を入力します。
4. 入力した単語を含むコマンドが表示されます。入力ごとに候補が絞られていきます。
5. 目的のコマンドが表示されたら、右クリックして **Assign Shortcut** を選択します。
![Command Palette Right-click](./img/command-palette-right-click.png)
6. 「Press a key to assign shortcut...」と表示されたら、割り当てたいキーを押します。
7. キーボードショートカットが割り当てられます。
8. ショートカットを削除するには、コマンドパレットのコマンド名を右クリックし、**Remove Shortcut**を選択します。

<Callout type="warning" emoji="⚠️">
すでに使用されているキーボードショートカットを割り当てた場合、警告メッセージなしに上書きされます。重複を避けるために、<kbd>Alt</kbd>、 <kbd>Ctrl</kbd>、 <kbd>Shift</kbd>などのキーを組み合わせてください。
</Callout>

## お気に入りの登録
1. <kbd>F</kbd>キーを押します。
2. コマンドパレットが表示されます。
3. コマンド入力欄にコマンド名を入力します。
4. 入力した単語を含むコマンドが表示されます。入力ごとに候補が絞られていきます。
5. 目的のコマンドが表示されたら、右クリックして **Add to favorites** を選択します。
![Command Palette Right-click](./img/command-palette-right-click.png)
6. コマンドがお気に入りに追加され、入力フィールドの下にボタンが表示されます。
7. お気に入りからコマンドを削除するには、お気に入りアイコンまたはコマンドパレットのコマンド名を右クリックし、**Remove from Favorites**を選択します。
