# Selecting Objects

## Selecting with the mouse

- **Click:** Select the object.
- **Drag from right to left:** Select objects that touch the green selection.
- **Drag from left to right:** Select objects completely contained in the purple selection.

![Mouse Drag selection](/img/mouse-drag-selection.gif)

## Selecting with Shortcuts

| Command Name      | Shortcut                    | Note                                                    |
| ----------------- | --------------------------- | ------------------------------------------------------- |
| Select all        | <kbd>A</kbd>                | Select all objects.                                             |
| Select all Curves | <kbd>Ctrl + A</kbd>         | Select all Curves.                                      |
| Deselect all      | <kbd>Escape</kbd>           | Deselect all.                                           |
| Invert Selection  | <kbd>Alt + A</kbd>          | Invert selection.                                       |
| Select Adjacent   | <kbd>Ctrl + Shift + =</kbd> | Select adjacent objects to the currently selected ones. |

| Note                                         | Shortcut                      |
| -------------------------------------------- | ----------------------------- |
| Select Edge loops                            | <kbd>Alt</kbd> + click        |
| Select Edge rings                            | <kbd>Alt + Ctrl</kbd> + click |
| Select all Curves, Regions on the same plane | <kbd>Alt</kbd> + click        |
| Select all parallel Faces                    | <kbd>Alt + Ctrl</kbd> + click |

### Select from the currently selected objects

| Command Name                | Shortcut            | Note                                                                        |
| --------------------------- | ------------------- | --------------------------------------------------------------------------- |
| Selection: convert to edges  | <kbd>Ctrl + 2</kbd> | Select only the Edges of the currently selected Solids, Sheets, and Faces.  |
| selection: convert to faces  | <kbd>Ctrl + 3</kbd> | Select only the Faces of the currently selected Solids and Sheets.          |
| selection: convert to solids | <kbd>Ctrl + 4</kbd> | Select only the Solids or Sheets of the currently selected Faces and Edges. |
| selection: convert to groups| <kbd>Ctrl + 5</kbd> |Select all objects within the group of the selected objects.|

![Select objects included](/img/select-objects-included.gif)

### Select the source objects used in the last operation

| Command Name                      | Shortcut                  | Note                                                                                                                                          |
| --------------------------------- | ------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Select Previous Entity Collection | <kbd>Ctrl + T</kbd>       | Select the source objects used in the last operation. Using this feature multiple times selects sources from earlier operations sequentially. |
| Select Next Entity Collection     | <kbd>Ctrl + Shift + T</kbd> | A counterpart to "Select Previous Entity Collection". Allows for the sequential selection of subsequent source objects.                       |
