# P-Menü

Das P-Menü kann geöffnet werden, indem Sie auf das P oben links klicken.

![P-Menü](/img/file-menu.png)

| Befehlsname          | Tastenkürzel                    | Funktion                                                                                                                                                                                         |
| --------------------- | ------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| New                   | <kbd>Strg + N</kbd>             | Löscht die Szene und öffnet ein neues Dokument.                                                                                                                                                   |
| New Window            |                                 | Startet Plasticity und öffnet ein neues Fenster. Die Akzentfarbe des neuen Fensters wird zufällig geändert, um es vom Hauptfenster zu unterscheiden. Kopieren und Einfügen zwischen Fenstern ist ebenfalls möglich. |
| Open                  | <kbd>Strg + O</kbd>             | Öffnet eine Plasticity-Datei.                                                                                                                                                                     |
| Import/Append         | <kbd>Strg + Umschalt + O</kbd>  | Importiert Dateien. Siehe [hier](import-export#file-import) für importierbare Formate.                                                                                                          |
| Save                  | <kbd>Strg + S</kbd>             | Speichert das aktuelle Dokument.                                                                                                                                                                  |
| Save as               | <kbd>Strg + Umschalt + S</kbd>  | Speichert das Dokument unter einem neuen Namen.                                                                                                                                                   |
| Export                | <kbd>Strg + Umschalt + E</kbd>  | Exportiert Dateien. Siehe [hier](import-export#file-export) für exportierbare Formate.                                                                                                          |
| Quick save version    |                                 | Speichert eine Kopie der Datei mit einem Zeitstempel im Dateinamen.                                                                                                                               |
| Save as startup scene |                             | Speichert das aktuelle Dokument als Startszene, die beim nächsten Start automatisch geöffnet wird.                                                                                                |
| Preferences           | <kbd>Strg + , (Komma)</kbd>     | Siehe [Einstellungen](preferences).                                                                                                                                                             |
| Manage licenses       |                                 | Verwaltet Lizenztokens und Geräte.                                                                                                                                                                |
| Plasticity Version    |                                 | Zeigt die installierte Version von Plasticity.                                                                                                                                                     |
| Quit                  | <kbd>Strg + Q</kbd>             | Beendet Plasticity.                                                                                                                                                                               |
