# 選択モード

選択したいオブジェクトに応じて選択モードを切り替えます。
選択モードの切り替えは、キーボードショートカットか画面左上のボタンでできます。

![Selection mode buttons](/img/selection-mode-buttons.png)


| Command Name                      | Shortcut             | Button                       | Note                                                         |
| --------------------------------- | -------------------- | ---------------------------- | ------------------------------------------------------------ |
| Selection mode: set Control Point | <kbd>1</kbd>         | (1)                          | 制御点選択モード（制御点が選択できる）                       |
| Selection mode: set edge          | <kbd>2</kbd>         | (2)                          | エッジ選択モード（エッジとカーブが選択できる）               |
| Selection mode: set face          | <kbd>3</kbd>         | (3)                          | フェイス選択モード（フェイスとリージョンが選択できる）       |
| Selection mode: set solid         | <kbd>4</kbd>         | (4)                          | ソリッド選択モード（ソリッドとシートが選択できる）           |
| Selection mode: set all           | <kbd>Tab</kbd>       | (5)                          | オール選択モード（すべてのタイプのオブジェクトが選択できる） |
| Selection: toggle Control Point   | <kbd>Shift + 1</kbd> | <kbd>Shift</kbd> + click (1) | 制御点選択モードのみをオン・オフする                         |
| Selection: toggle edge            | <kbd>Shift + 2</kbd> | <kbd>Shift</kbd> + click (2) | エッジ選択モードのみをオン・オフする                         |
| Selection: toggle face            | <kbd>Shift + 3</kbd> | <kbd>Shift</kbd> + click (3) | フェイス選択モードのみをオン・オフする                       |
| Slection: toggle solid            | <kbd>Shift + 4</kbd> | <kbd>Shift</kbd> + click (4) | ボディ選択モードのみをオン・オフする                         |
