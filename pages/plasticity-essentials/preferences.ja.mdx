# 環境設定

環境設定ウィンドウを開くには、左上隅にあるPメニューをクリックし、**Preferences**を選択します。
設定を変更したときは、Plasticityを再起動する必要があります。

## Navigation（ナビゲーション）

### Navigation settings（ナビゲーション設定）

- **Zoom to cursor** : 
    - 有効にすると、カーソル位置にズームします。 (デフォルト: 有効)
- **Rotate around cursor** : 
    - 有効にすると、カーソル位置でビューが回転します。 (デフォルト: 有効)
- **Zoom vertical** : 
    - 有効にすると、<kbd>Ctrl + 中ボタン</kbd>の垂直ドラッグでズームイン/アウトします。 (デフォルト: 有効)
- **Invert zoom direction** : 
    - 有効にすると、ズームの方向が反転します。 (デフォルト: 無効)
- **Invert wheel direction** : 
    - 有効にすると、マウスホイールの方向が反転します。 (デフォルト: 無効)
    
### Navigation Presets（ナビゲーションプリセット）

- 3Dビューの操作方法を他の3Dソフトウェアと同じにすることができます。

## Performance（パフォーマンス）

- GPUレンダリング品質とCPUファセット品質を指定できます。

## Appearance（外観）

- アクセントカラー、ニュートラルカラー、ビューポートの背景色を選択できます。

## Grid & Units（グリッドと単位）

- **Units:** ビューポートの単位を指定します。
- **Grid size:** グリッド全体の大きさを指定します。
- **Accented lines every:** グリッドの太い線が描画される間隔を指定します。
- **Show measurements when drawing Curves:** 有効にすると、カーブを描くときに数値が表示されます。数値を入力するには<kbd>Tab</kbd>を押します。

## Backup（バックアップ）

- 緊急時に備えてバックアップをとっておくことができます。

## Server（サーバー）

- Blender Bridgeで使用するサーバーを設定できます。
- デフォルトは、Host: localhost、 Port: 8980 です。この値をBlenderのアドオンでの設定と合わせてください。

## SpaceMouse（スペースマウス）

- 3DconnexionのSpaceMouseを使用する場合の設定項目です。
- WindowsとmacOSでの動作確認済みは、SpaceMouse CompactとSpaceMouse Pro、SpaceMouse Enterpriseです。Linuxは動作未確認です。

## General（一般）

- Automatic invoke commands like offset, extrude, and fillet, based on the current selection.
    - 有効にすると、現在の選択に基づいて、オフセット、押し出し、フィレットなどのコマンドを自動的に呼び出します（デフォルト：有効）。
