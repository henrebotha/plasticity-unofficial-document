# オブジェクトステイタスの変更

- オブジェクトやグループのステイタスを変更することができます。
    - 一時的に非表示にしたり、再び表示したりすることができます。
    - 一時的に無効にしたり、再び有効にしたりできます。
    - ロックまたはアンロックできます。
![Toggle Object States](/img/toggle-object-states.png)

### アウトライナーでの操作

- <ButtonIcon type="button-outliner-hide" double="true" />: 表示・非表示を切り替えます。
- <ButtonIcon type="button-outliner-disable" double="true" /> : 使用可否を切り替えます。
- <ButtonIcon type="button-outliner-disable-selection" double="true"  /> : ロックを切り替えます。

### ショートカットキーでの操作

| 操作                                 | ショートカットキーでの操作     | コマンド名      |
| ------------------------------------ | ------------------------------ | --------------- |
| 選択しているオブジェクトを隠す       | <kbd>H</kbd>                   | Hide selected   |
| 選択されていないオブジェクトを隠す   | <kbd>Shift + H</kbd>           | Hide unselected |
| すべての表示する                     | <kbd>Alt + H</kbd>             | Unhide all      |
| 非表示を反転する                     | <kbd>Ctrl + H</kbd>            | Invert Hidden   |
| 同じ平面のすべてのカーブを隠す       | Select Regions -> <kbd>H</kbd> |                 |
| 選択しているオブジェクトをロックする |                                | Lock selected   |
