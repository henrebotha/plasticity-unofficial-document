# Gitter und Fang

## Gitter[#grid]

- Das Gitter wird auf der aktiven Konstruktionsebene angezeigt. Sie können die Gittereinheit und -größe in den [preferences](/plasticity-essentials/preferences) festlegen.
- Sie können das Fang am Gitter verwenden, um Ihren Cursor beim Zeichnen oder Verschieben von Objekten am Gitter auszurichten.
- Um das Fang am Gitter umzuschalten, klicken Sie in der oberen rechten Ecke des Fensters auf **Snap to grid**. Um die Gittergröße festzulegen, ändern Sie den Wert unter **Snap to grid**.
- Um das Fang am Gitter vorübergehend zu deaktivieren, halten Sie <kbd>Strg</kbd> während des Ziehens gedrückt.
- Sie können das Fang am Gitter im Befehlspalette ausführen; **Grid: toggle grid snapping**, **Viewport: decrease grid size**, **Viewport: increase grid size**.

## Objekt Fang[#object-snap]

- Sie können das Einrasten verwenden, um Ihren Cursor beim Zeichnen oder Verschieben von Objekten an Merkmale auszurichten.
- Um das Einrasten von Objekten umzuschalten, klicken Sie in der oberen rechten Ecke des Fensters auf **Enable snapping**. Um verschiedene Einrasttypen umzuschalten, klicken Sie auf <ButtonIcon type="button-types-of-snapping" /> und passen Sie die Optionen an.
- Um das Einrasten vorübergehend zu aktivieren, halten Sie die <kbd>Strg</kbd> Taste gedrückt. Beim Zeichnen von Kurven wird das Einrasten automatisch aktiviert.
- Sie können an Punkten verschiedener Objekttypen ausrichten: Control-Vertex, Curve, Region, Edge, and Face. Ein Fangziel erscheint als kleiner schwarzer Punkt.
- Ein Hinweistext erscheint, wenn Sie auf ein Fangziel zeigen. Klicken Sie, wenn der Hinweistext sichtbar ist, um genau auszurichten.

    | Fangziel       | Hinweistext                                                                                                                  |
    | -------------- | ---------------------------------------------------------------------------------------------------------------------------- |
    | Control-Vertex | CV                                                                                                                           |
    | Curve          | Curve, Beginning, End, Middle, Closest, Intersection, X, Y, Z, XY, YZ, ZX, Center, 1/4, 3/4, Perpendicular, Tangent, Tan/Tan |
    | Region         | Region, Center                                                                                                               |
    | Edge           | Edge, End, Middle                                                                                                            |
    | Face           | Face, Center                                                                                                                 |

- Sie können das Einrasten von Objekten im Befehlspalette ausführen; **Snaps: toggle snap snapping**.
