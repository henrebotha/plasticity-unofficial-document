# Group Objects

You can organize objects by grouping them.

| Operation                                                                         | Shortcut                         | Command Name     |
| --------------------------------------------------------------------------------- | -------------------------------- | ---------------- |
| Create a group, and the selected object will be moved to the newly created group. | <kbd>Ctrl + G</kbd> or click (1) | Group Selected   |
| Remove the selected object from the group.                                        | <kbd>Alt + G</kbd>               | Ungroup Selected |
| Collapse all groups                                                               | Click (2)                        |                  |
