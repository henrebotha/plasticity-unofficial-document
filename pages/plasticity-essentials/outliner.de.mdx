# Gliederung

Das Gliederung zeigt alle von Ihnen erstellten Objekte an. Im Gliederung können Sie Objekte organisieren, gruppieren, ihren Zustand wechseln und Materialien konfigurieren.

![Gliederung](/img/outliner-001.png)

## Angezeigte Objekte

- Die von Ihnen erstellten Objekte werden in verschiedene Bereiche eingeteilt: "Solids", "Sheets", "Curves", "Empties" und "Measurements".
- Unter "Empties" finden Sie importierte Bilder und Meshes, und unter "Measurements" werden die Maße angezeigt.

## Objekte organisieren

- Klicken Sie mit der rechten Maustaste auf eine Abschnittsbezeichnung ("Solids," "Sheets," "Curves") und wählen Sie "Select all", um alle Objekte in diesem Abschnitt auszuwählen.
- Sie können das Gliederung durch Ziehen und Ablegen von Objekten neu organisieren.

## Objekte gruppieren

Gruppieren ermöglicht es Ihnen, Ihre Objekte besser zu organisieren.

| Vorgang                                                                        | Tastenkombination                                                  | Befehlsname      |
| ------------------------------------------------------------------------------ | ------------------------------------------------------------------ | ---------------- |
| Eine Gruppe erstellen und das ausgewählte Objekt zur neuen Gruppe verschieben. | <kbd>Strg + G</kbd> oder <ButtonIcon type="outliner-gruop-icon" /> | Group Selected   |
| Das ausgewählte Objekt aus der Gruppe entfernen.                               | <kbd>Alt + G</kbd>                                                 | Ungroup Selected |

Klicken Sie mit der rechten Maustaste auf einen Gruppenordner, um auf zugehörige Befehle zuzugreifen.

![Gruppenobjekte-Menü](/img/outliner-group-objects-menu.png)
| Menü                                             | Anmerkung                                                                     |
| ------------------------------------------------ | ----------------------------------------------------------------------------- |
| Aktuelle Auswahl zur Gruppe hinzufügen           | Fügt das ausgewählte Objekt zur Gruppe hinzu.                                 |
| Alles auswählen                                  | Wählt alle Objekte innerhalb der Gruppe aus.                                  |
| Neue Untergruppe mit aktueller Auswahl erstellen | Erstellt eine neue Untergruppe und fügt das ausgewählte Objekt hinzu.         |
| Alles einklappen                                 | Klappen Sie alle Gruppen ein.                                                 |
| Alles ausklappen                                 | Klappen Sie alle Gruppen aus.                                                 |
| Material hinzufügen                              | Weist der Gruppe ein Material zu. Siehe [Materialfarbe](#material-color).     |
| Material entfernen                               | Entfernt das Material aus der Gruppe. Siehe [Materialfarbe](#material-color). |

Alt-Klick auf einen Ordner im Gliederung, um alle Objekte in diesem Abschnitt auszuwählen.

### Gruppen einklappen und ausklappen

![Gruppen einklappen und ausklappen](/img/outliner-collapse-expand.png)

- Um alle Gruppen einzuklappen, klicken Sie entweder auf (1) oder bewegen Sie den Cursor über das Gliederung und drücken Sie <kbd>C</kbd>.
- Um alle Gruppen auszuklappen, klicken Sie auf (2) und wählen Sie "Alles ausklappen" aus dem Dropdown-Menü.

## Objekte umbenennen

- Um ein Objekt umzubenennen, doppelklicken Sie auf seinen Namen.
- Um mehrere Objekte gleichzeitig umzubenennen, halten Sie die Umschalt-Taste gedrückt, während Sie sie auswählen, und doppelklicken Sie dann, während Sie Umschalt gedrückt halten, auf einen ihrer Namen.

## Objekte im Gliederung suchen

![Objekte im Gliederung suchen](/img/outliner-collapse-expand.png)

### Um ein ausgewähltes Objekt an den Anfang der Gliederung zu bringen

- Wählen Sie das Objekt im 3D-Viewport aus, bewegen Sie den Cursor über das Gliederung und drücken Sie <kbd>/</kbd>.
- Alternativ können Sie das Objekt auswählen, auf (2) klicken und "Im Gliederung auswählen" wählen.

### Um ein ausgewähltes Objekt im Gliederung zu isolieren

- Wählen Sie das Objekt im Gliederung aus und drücken Sie <kbd>.</kbd>. Dies klappt alle Gruppen und Abschnitte ein, außer dem, der Ihr ausgewähltes Objekt enthält.
- Alternativ können Sie das Objekt auswählen, auf (2) klicken und "Auswahl isolieren" wählen.

## Objektzustände wechseln

Sie können die Zustände von Objekten oder Gruppen wechseln, z. B. sie vorübergehend ausblenden oder anzeigen, deaktivieren oder aktivieren und sperren oder entsperren.

![Objektzustände wechseln](/img/toggle-object-states.png)

### Vorgänge im Gliederung

Klicken Sie auf die Symbole neben Objekten oder Gruppen, um ihre Zustände zu ändern. Sie können Zustände auch in großen Mengen ändern, indem Sie mehrere Objekte auswählen.

- <ButtonIcon type="button-outliner-hide" double="true" />: Ausblenden wechseln
- <ButtonIcon type="button-outliner-disable" double="true" /> : Deaktivieren wechseln
- <ButtonIcon type="button-outliner-disable-selection" double="true"  /> : Sperren wechseln

### Vorgänge durch Tastenkombination

Sie können auch Tastenkombinationen verwenden, um die Zustände von Objekten oder Gruppen zu ändern.

| Vorgang                                    | Tastenkombination                  | Befehlsname     |
| ------------------------------------------ | ---------------------------------- | --------------- |
| Ausgewählte Objekte ausblenden             | <kbd>H</kbd>                       | Hide selected   |
| Nicht ausgewählte Objekte ausblenden       | <kbd>Umschalt + H</kbd>            | Hide unselected |
| Alle einblenden                            | <kbd>Alt + H</kbd>                 | Unhide all      |
| Ausgeblendete umkehren                     | <kbd>Strg + H</kbd>                | Invert Hidden   |
| Alle Curves auf derselben Ebene ausblenden | Regionen auswählen -> <kbd>H</kbd> |                 |
| Ausgewählte Objekte sperren                |                                    | Lock selected   |

## Materialfarbe

Sie können Materialien Solids, Sheets und Curves zuweisen. Für eine Gruppe eingestellte Materialien werden auch auf Objekte innerhalb dieser Gruppe angewendet.
Weitere Informationen zu Materialien finden Sie unter [**Set Material**](/common/set-material) und [**Remove Material**](/common/remove-material).
![Gliederung 4](/img/outliner-04.png)

Im Gliederung können Sie:

- Die Materialfarbe des Objekts wird als Farbpunkte angezeigt.
- Die Farbe kann durch Ziehen und Ablegen der Farbpunkte kopiert werden.
- Materialfarben können auch für Gruppen festgelegt werden. Die Farbe kann auf die gleiche Weise kopiert werden.
- Klicken Sie mit der rechten Maustaste auf ein Objekt, um ein Menü aufzurufen, in dem Sie Materialien festlegen oder entfernen können.
- Sie können auch mehrere Objekte auswählen und Materialien in großen Mengen festlegen oder entfernen.
