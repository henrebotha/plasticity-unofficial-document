# Einstellungen

Um das Einstellungsfenster zu öffnen, klicken Sie auf das P-Menü in der oberen linken Ecke und wählen Sie **Preferences**.
Sie müssen Plasticity neu starten, wenn Sie die Einstellungen ändern.

## Navigation

### Navigation Settings

- **Zoom to cursor**:
    - Bei Aktivierung wird am Cursor positioniert gezoomt. (Standard: Aktiviert)
- **Rotate around cursor**:
    - Bei Aktivierung dreht sich die Ansicht um die Cursorposition. (Standard: Aktiviert)
- **Zoom vertical**:
    - Bei Aktivierung erfolgt das Zoomen durch vertikales Ziehen mit <kbd>Strg + Mitteltaste</kbd>. (Standard: Aktiviert)
- **Invert zoom direction**:
    - Bei Aktivierung wird die Zoomrichtung umgekehrt. (Standard: Deaktiviert)
- **Invert wheel direction**:
    - Bei Aktivierung wird die Richtung des Mausrads umgekehrt. (Standard: Deaktiviert)

### Navigation Presets

- Sie können Voreinstellungen für dieselben Steuerungen wie bei anderer 3D-Software auswählen.

## Performance

- Sie können die GPU-Renderingqualität und die CPU-Facettenqualität festlegen.

## Appearance

- Sie können Akzentfarben, neutrale Farben und Hintergrundfarben des Viewports auswählen.

## Grid & Units

- **Units:** Legt die Einheiten für den Viewport fest.
- **Grid size:** Legt die Gesamtgröße des Rasters fest.
- **Accented lines every:** Legt das Intervall fest, in dem dicke Linien des Rasters angezeigt werden.
- **Show measurements when drawing Curves:** Bei Aktivierung werden Messungen beim Zeichnen von Kurven angezeigt. Drücken Sie <kbd>Tab</kbd>, um Werte einzugeben.

## Backup

- Sie können Backups zur Vorbereitung auf Notfälle erstellen.

## Server

- Konfigurieren Sie den Server, der mit der Blender Bridge verwendet werden soll.
- Standard ist Host: localhost, Port: 8980. Passen Sie diese Werte an die Einstellungen im Blender-Addon an.

## SpaceMouse

- Dies sind die Einstellungen für die Verwendung der 3Dconnexion SpaceMouse.
- Die SpaceMouse Compact, SpaceMouse Pro und SpaceMouse Enterprise wurden für die Funktion auf Windows und macOS bestätigt. Die Funktionsweise auf Linux ist unbestätigt.

## General

- Automatic invoke commands like offset, extrude, and fillet, based on the current selection.
    - Bei Aktivierung werden automatisch Befehle wie Versatz, Extrusion und Fase basierend auf der aktuellen Auswahl aufgerufen (Standard: Aktiviert).
