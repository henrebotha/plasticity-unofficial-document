# Einstellungen

Um das Einstellungsfenster zu öffnen, klicken Sie auf das P in der oberen linken Ecke des Hauptfensters und wählen Sie **Preferences** aus dem Menü.

## Navigation

### Navigationseinstellungen

- **Zoom to cursor** : 
    - Wenn aktiviert, wird die Ansicht auf die Cursorposition gezoomt. (Standard: aktiviert)
- **Rotate around cursor** : 
    - Wenn aktiviert, wird die Ansicht um die Cursorposition gedreht. (Standard: aktiviert)
- **Zoom vertical** : 
    - Wenn aktiviert, wird durch vertikales Ziehen mit <kbd>Strg + mittlerer Maustaste</kbd> hinein- und herausgezoomt. (Standard: aktiviert)
- **Invert zoom direction** : 
    - Wenn aktiviert, wird die Zoomrichtung umgekehrt. (Standard: deaktiviert)
- **Invert wheel direction** : 
    - Wenn aktiviert, wird die Richtung des Rades umgekehrt. (Standard: deaktiviert)


### Navigation presets

- Sie können Voreinstellungen für dieselben Steuerelemente wie andere 3D-Software wählen.

## Performance

- Sie können die GPU-Rendering-Qualität und die CPUFaceting-Qualität anpassen.

## Appearance

- Sie können die Akzentfarbe, die neutrale Farbe und die Hintergrundfarbe des Viewports festlegen.

## Grid & Units

- **Units:** Geben Sie die Einheiten für den Viewport an.
- **Grid size:** Geben Sie die Größe des Bereichs an, in dem die Gitterlinien gezeichnet werden.
- **Accented lines every:** Geben Sie den Abstand der dicken Linien des Gitters an.

- **Show measurements when drawing Curves:** Wenn aktiviert, werden numerische Werte beim Zeichnen von Kurven angezeigt. Sie können <kbd>Tab</kbd> drücken, um Werte einzugeben.

## Backup

- Sie können Sicherungen für den Notfall aufbewahren.
