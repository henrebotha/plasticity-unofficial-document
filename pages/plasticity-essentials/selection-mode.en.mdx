# Selection Mode

Switch the selection mode according to the object you want to select.
To switch the selection mode, use the buttons in the upper left corner of the screen or keyboard shortcuts.

![Selection mode buttons](/img/selection-mode-buttons.png)

| Command Name                      | Shortcut             | Button                       | Note                                      |
| --------------------------------- | -------------------- | ---------------------------- | ----------------------------------------- |
| Selection mode: set Control Point | <kbd>1</kbd>         | (1)                          | Allows Control-Point selection.           |
| Selection mode: set edge          | <kbd>2</kbd>         | (2)                          | Allows edge or curve selection.           |
| Selection mode: set face          | <kbd>3</kbd>         | (3)                          | Allows face or region selection.          |
| Selection mode: set solid         | <kbd>4</kbd>         | (4)                          | Allows solid or sheet selection.          |
| Selection mode: set all           | <kbd>Tab</kbd>       | (5)                          | Allows selection of all types of objects. |
| Selection: toggle Control Point   | <kbd>Shift + 1</kbd> | <kbd>Shift</kbd> + click (1) | Toggle Control Point selection mode.      |
| Selection: toggle edge            | <kbd>Shift + 2</kbd> | <kbd>Shift</kbd> + click (2) | Toggle Edge selection mode.               |
| Selection: toggle face            | <kbd>Shift + 3</kbd> | <kbd>Shift</kbd> + click (3) | Toggle Face selection mode.               |
| Slection: toggle solid            | <kbd>Shift + 4</kbd> | <kbd>Shift</kbd> + click (4) | Toggle Body selection mode.               |
|                                   |                      |                              |                                           |
