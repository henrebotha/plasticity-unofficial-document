## Befehle für Festkörper

| Befehlsname                                                  | Tastenkombination            | Anmerkung                                                                                                                  |
| ------------------------------------------------------------ | ---------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| [Boolean](solid/boolean)                                     | <kbd>Q</kbd>                 | Vereinigung von Festkörpern und/oder Blättern.                                                                             |
| [Cut](solid/cut)                                             | <kbd>C</kbd>                 | Festkörper und Blatt schneiden.                                                                                            |
| **Befehle für Festkörper und Blätter**                       |                              |                                                                                                                            |
| [Extrude](solid/extrude)                                     | <kbd>E</kbd>                 | Gesichter, Regionen und Kurven extrudieren.                                                                                |
| [Sweep](solid/sweep)                                         | <kbd>Umschalt + P</kbd>         | Region oder Kurve kehren.                                                                                                  |
| [Revolve](solid/revolve)                                     |                              | Region oder Kurve rotieren.                                                                                                |
| [Loft](solid/loft)                                           | <kbd>L</kbd>                 | Gesicht, Kante und Kurve loftieren.                                                                                        |
| [Pipe](solid/pipe)                                           | <kbd>P</kbd>                 | Erstelle ein Rohr aus Kante und Kurve.                                                                                     |
| [Patch](solid/patch)                                         |                              | Löcher im Blatt füllen / Geschlossene Kurven zu einem Blatt.                                                               |
| [Thicken](solid/thicken)                                     |                              | Einem ausgewählten Blatt Dicke hinzufügen.                                                                                 |
| [Thicken Face](solid/thicken-face)                           |                              | Einem ausgewählten Gesicht Dicke hinzufügen.                                                                               |
| [Constrained Surface](solid/constrained-surface)             |                              | Eine eingeschränkte Oberfläche erstellen.                                                                                  |
| [Fillet](solid/fillet)                                       | <kbd>B</kbd>                 | Vereinigter Befehl für Kurvenabrundung, Scheitelpunktabrundung und Schalenabrundung.                                       |
| [Fillet Shell](solid/fillet-shell)                           | <kbd>B</kbd>                 | Kanten abrunden oder abschrägen.                                                                                           |
| [Refillet Face](solid/refillet-face)                         |                              | Abrundungen erneut ausführen.                                                                                              |
| [Remove Fillets from Shell](solid/remove-fillets-from-shell) |                              | Abrundungen von Festkörpern und Blättern entfernen.                                                                        |
| [Alternative Duplicate](solid/alternative-duplicate)         | <kbd>Alt + D</kbd>           | Alternatives Duplizieren.                                                                                                  |
| [Hollow](solid/hollow)                                       |                              | Einen Festkörper aushöhlen.                                                                                                |
| [Rebuild](solid/rebuild)                                     |                              | Gesicht neu erstellen.                                                                                                     |
| **Blattbefehle**                                             |                              |                                                                                                                            |
| [Join](solid/join)                                           | <kbd>J</kbd>                 | Vereinigter Befehl zum Verbinden von Kurven und Blättern.                                                                  |
| [Join Sheets](solid/join-sheets)                             | <kbd>J</kbd>                 | Blätter mit übereinstimmenden Kanten verbinden.                                                                            |
| [Bridge Surface](solid/bridge-surface)                       |                              | Zwei Blätter überbrücken.                                                                                                  |
| [Extend](solid/extend)                                       |                              | Vereinigter Befehl zum Erweitern von Kurven und Blättern.                                                                  |
| [Extend Sheet](solid/extend-sheet)                           |                              | Eine oder mehrere äußere Kanten des Blatts erweitern.                                                                      |
| [Unjoin](solid/unjoin)                                       | <kbd>Alt + J</kbd>           | Vereinigter Befehl zum Trennen von Kurven, Trennen von Gesichtern, Trennen von Schalen.                                    |
| [Unjoin Faces](solid/unjoin-faces)                           | <kbd>Alt + J</kbd>           | Ausgewählte Gesichter von Festkörpern oder Blättern trennen.                                                               |
| [Unjoin Shells](solid/unjoin-shells)                         | <kbd>Alt + J</kbd>           | Alle Gesichter von ausgewählten Festkörpern oder Blättern in einzelne Gesichter trennen.                                   |
| [Find Boundary Edges](solid/find-boundary-edges)             |                              | Wähle Kanten einer Blattöffnung aus.                                                                                       |
| [Untrim](solid/untrim)                                       | <kbd>Alt + T</kbd>           | Ein Blatt basierend auf der ursprünglichen Form erstellen.                                                                 |
| **Gesichtsbefehle**                                          |                              |                                                                                                                            |
| [Offset Face](solid/offset-face)                             | <kbd>Umschalt + O</kbd>         | Die ausgewählten Gesichter versetzen.                                                                                      |
| [Draft Face](solid/draft-face)                               |                              | Gesicht kippen.                                                                                                            |
| [Match Face](solid/match-face)                               |                              | Gesichter an ein anderes Gesicht anpassen.                                                                                 |
| [Delete Face](solid/delete-face)                             | <kbd>Umschalt + Backspace</kbd> | Gesicht löschen.                                                                                                           |
| [Unwrap Face](solid/unwrap-face)                             |                              | Gesicht auspacken.                                                                                                         |
| [Wrap Face](solid/wrap-face)                                 |                              | Gesicht einpacken.                                                                                                         |
| **Kantenbefehle**                                            |                              |                                                                                                                            |
| [Offset Curve](solid/offset-curve)                           | <kbd>O</kbd>                 | Vereinigter Befehl für Versatz einer ebenen Kurve, Versatz einer Region, Versatz einer Gesichtsschlaufe und Kantenversatz. |
| [Offset Edge](solid/offset-edge)                             | <kbd>O</kbd>                 | Die ausgewählten Kanten versetzen.                                                                                         |
| [Offset Face Loop](solid/offset-face-loop)                   | <kbd>O</kbd>                 | Die Umrisskanten von ausgewählten Gesichtern versetzen.                                                                    |
| [Imprint](solid/imprint)                                     | <kbd>Umschalt + I</kbd>         | Vereinigter Befehl für Prägung Körper-Körper, Prägung Kurve-Körper.                                                        |
| [Imprint Body Body](solid/imprint-body-body)                 | <kbd>Umschalt + I</kbd>         | Kanten auf dem Ziel-Festkörper oder -Blatt erstellen, wo es mit einem Werkzeug-Festkörper oder -Blatt schneidet.           |
| [Imprint Curve Body](solid/imprint-curve-body)               | <kbd>Umschalt + I</kbd>         | Kanten auf dem Ziel-Festkörper oder -Blatt erstellen, wo es auf eine Werkzeug-Kurve projiziert wird.                       |
| [Isoparam](solid/isoparam)                                   | <kbd>Strg + R</kbd>          | Isoparams hinzufügen.                                                                                                      |
| [Delete Redundant Topology](solid/delete-redundant-topology) |                              | Überflüssige Kanten entfernen.                                                                                             |
| [Extend Edge](solid/extend-edge)                             |                              | Kanten erweitern.                                                                                                          |
