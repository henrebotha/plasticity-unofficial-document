# Tangent Circle 

<Reference shortcut="" command="Tangent Circle" button="Toolbar" />

Einen Kreis erstellen, der tangential zu zwei Kurven mit einem bestimmten Radius ist.

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-tangent-circle" /> in der Symbolleiste.
2. Klicken Sie, um zwei Kurvensegmente auszuwählen, die tangential zu einem Kreis sein sollen.
3. Geben Sie den Radius an, indem Sie den Cursor bewegen oder einen Wert eingeben.
- \[ Befehlsdialog \]
    - Select first curve segment: Wenn hervorgehoben, klicken Sie, um das erste Kurvensegment auszuwählen.
    - Select second curve segment: Wenn hervorgehoben, klicken Sie, um das zweite Kurvensegment auszuwählen.
    - Radius: Geben Sie den Radius des Kreises ein.
4. Klicken Sie, um den Radius zu bestätigen.
5. Wenn Sie einen Wert eingeben, bestätigen Sie mit OK im Befehlsdialog oder per Rechtsklick.
6. Ein Kreis wird erstellt.
