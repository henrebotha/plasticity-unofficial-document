# Center Point Arc

<Reference shortcut='' command='Center Point Arc' button='Werkzeugleiste' />

Sie können einen Bogen erstellen, indem Sie das Zentrum und zwei Punkte auf dem Umfang angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-center-point-arc" /> Knopf in der Werkzeugleiste.
2. Klicken Sie auf den Mittelpunkt des Bogens.
3. Bewegen Sie den Cursor und klicken Sie auf die erste Position des Bogens.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
4. Bewegen Sie den Cursor, um die Größe des Bogens anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>K</kbd> **Knife:** Bitte beachten Sie [Messermodus verwenden](/tool/sketching-essentials#use-knife-mode).
      - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Auf X, Y, Z Achse beschränken](/tool/sketching-essentials#constrain-to-an-axis).
5. Der Bogen wird erstellt.
