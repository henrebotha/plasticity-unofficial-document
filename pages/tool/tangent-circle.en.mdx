# Tangent Circle 

<Reference shortcut="" command="Tangent Circle" button="Toolbar" />

Create a circle tangent to two curves with a specified radius.

1. Click the button <ButtonIcon type="button-tangent-circle" /> on the toolbar.
2. Click to select two curve segments to be tangent to a circle.
3. Specify the radius by moving the cursol or entering a value.
- \[ Command Dialog \]
    - Select first curve segment : When highlighted, click to select the first curve segment.
    - Select second curve segment : When highlighted, click to select the second curve segment.
    - Radius: Enter the radius of the circle.
4. Click to conferm the radius.
5. When you enter the value for the radius, confirm with OK in the Command Dialog or right-click.
6. A circle is created.
