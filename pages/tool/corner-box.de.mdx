# Corner Box

<Reference shortcut='Umschalt + B' command='Corner Box' button='Werkzeugleiste' />

Erstellen Sie ein rechteckiges Kästchen, indem Sie diagonale Eckpunkte und Höhe angeben. Boolesche Operationen sind ebenfalls verfügbar.

## Grundlegende Nutzung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-corner-box" /> in der Werkzeugleiste.
2. Klicken Sie auf den Startpunkt der Diagonalen auf der Basisfläche.
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>C</kbd> **Center/Corner:** Wechselt zu Center Box.
   - <kbd>S</kbd> **Square:** Begrenzt auf ein Quadrat.
3. Klicken Sie auf den Endpunkt der Diagonalen.
4. Bewegen Sie den Cursor und klicken Sie auf die gewünschte Höhe.
5. Legen Sie die Optionen im Befehlsdialog und den Tastaturkürzeln fest.
   - <Kst>Befehlsdialog</Kst>
   - **Select target bodies**
      - Diese Optionen haben dieselbe Funktion wie der Boolean-Befehl. Siehe [**Boolean**](../solid/boolean) Befehl für Details.
   - **Width:** Geben Sie die Breite an.
   - **Length:** Geben Sie die Länge an.
   - **Height:** Geben Sie die Höhe an.
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>Umschalt + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Umschalt + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - Diese Optionen haben dieselbe Funktion wie der Boolean-Befehl. Siehe [**Boolean**](../solid/boolean) Befehl für Details.
   - <kbd>H</kbd> **Height:** Bewegt den Cursor, um die Höhe anzugeben.
6. Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.
7. Ein rechteckiges Kästchen wird erstellt.
