# Three-Point Circle

<Reference shortcut='' command='Three-Point Circle' button='Werkzeugleiste' />

Sie können einen Kreis erstellen, indem Sie drei Punkte auf seinem Umfang angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-three-point-circle" /> Knopf in der Werkzeugleiste.
2. Klicken Sie auf einen Punkt auf dem Umfang des Kreises.
3. Bewegen Sie den Cursor und klicken Sie auf einen zweiten Punkt auf dem Umfang.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Länge numerisch angeben. Weitere Informationen finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
      - <kbd>K</kbd> **Knife:** Siehe [Messermodus verwenden](/tool/sketching-essentials#use-knife-mode).
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Axis:** [Beschränkt auf X, Y, Z Achse](/tool/sketching-essentials#constrain-to-an-axis).
4. Bewegen Sie den Cursor, um die Größe anzupassen, und klicken Sie zur Bestätigung.
5. Der Kreis wird erstellt.
