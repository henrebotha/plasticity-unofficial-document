# Cylinder

<Reference shortcut='Umschalt + C' command='Cylinder' button='Werkzeugleiste' />

Sie können einen Zylinder erstellen, indem Sie seinen Grundkreis und die Höhe angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-cylinder" /> Knopf in der Werkzeugleiste.
2. Klicken Sie, um die Mittelposition des Kreises festzulegen.
3. Bewegen Sie den Cursor, um die Größe des Grundkreises anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Details finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
      - <kbd>V</kbd> **Vertical/ Horizontal:** Ändert die Neigung relativ zur Konstruktionsebene.
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Axis:** [Beschränkt auf X, Y, Z Achse](/tool/sketching-essentials#constrain-to-an-axis).
4. Bewegen Sie den Cursor, um die Höhe anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Befehlsdialog</Kst>
      - **Select target bodies**
         - Funktioniert wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](/solid/boolean) Befehl.
      - **Radius:** Durchmesser angeben.
      - **Height:** Höhe angeben.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Umschalt + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Umschalt + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
         - Funktioniert wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](/solid/boolean) Befehl.
      - <kbd>F</kbd> **Radius:** Durchmesser durch Bewegen des Cursors anpassen.
      - <kbd>D</kbd> **Height:** Höhe durch Bewegen des Cursors anpassen.
5. Sie können auch die Form anpassen, indem Sie die gelben Punkte bewegen.
6. Klicken Sie im Befehlsdialog auf OK oder rechtsklick zur Bestätigung.
7. Der Zylinder wird erstellt.
