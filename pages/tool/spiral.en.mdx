# Spiral

<Reference shortcut='' command='Spiral' button='Toolbar' />

You can create a spiral by specifying its diameter and number of turns. 

## Basic Usage

1. Click the <ButtonIcon type="button-spiral" /> button on the toolbar.
2. Click to set the starting point of the spiral's axis.
3. Move the cursor and click to set the end point of the spiral's axis.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrains to X, Y, Z axis](./sketching-essentials#constrain-to-an-axis).
4. Move the cursor to adjust the diameter of the spiral, and click to confirm.
5. Set options using the command dialog and keyboard shortcuts.
   - <Kst>Command Dialog</Kst>
      - **Turns:** Specify the number of rotations.
      - **Radius:** Specify the diameter.
      - **Angle OR Pitch:** Specify the angle or pitch. The values are interrelated.
      - **Handedness:** Reverse the direction of rotation.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>A</kbd> **Angle:** Specify the angle by moving the mouse.
      - <kbd>D</kbd> **Length:** Specify the length of the axis by moving the mouse.
      - <kbd>R</kbd> **Radius:** Specify the diameter by moving the mouse. The values are interrelated.
4. You can also adjust the shape by moving the yellow dots and white circle.
6. Click OK in the command dialog or right-click to confirm.
7. The spiral is created.
