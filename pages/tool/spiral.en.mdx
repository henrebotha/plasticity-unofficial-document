# Spiral

<Reference shortcut='' command='Spiral' button='Toolbar' />

Create a spiral by specifying the diameter and number of turns. Angling options are also available.

1. Click the button <ButtonIcon type="button-spiral" /> on the toolbar.
2. Click at the start point of the spiral axis.
3. Move the cursor and click at the endpoint of the spiral axis.
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrain to the X, Y, or Z-axis](./sketching-essentials#constrain-to-an-axis).
4. Move the cursor and click at the point for the diameter.
5. Set the options in the Command Dialog and Keyboard Shortcuts.
   - **Turns:** Specify the number of rotations.
   - **Radius:** Specify the diameter.
   - **Angle OR Pitch:** Specify the angle or pitch. Each value is linked.
   - **Handedness:** Reverses the direction of rotation.
   - <kbd>A</kbd> **Angle:** Specify the angle with mouse movement.
   - <kbd>D</kbd> **Length:** Specify the length of the axis with mouse movement.
   - <kbd>R</kbd> **Radius:** Specify the diameter with mouse movement.
6. Confirm with OK in the Command Dialog or right-click.
7. A spiral is created.
