# Tangent Circle 

<Reference shortcut="" command="Tangent Circle" button="Toolbar" />

2つのカーブに接する円を作成できます。

## 基本的な使い方

1. ツールバーの <ButtonIcon type="button-tangent-circle" /> ボタンをクリックします。
2. 円に接するカーブをクリックします。もうひとつのカーブもクリックします。
3. カーソルを動かして円の大きさを調整します。
    - <Kst>コマンドダイアログ</Kst>
        - **Select first curve segment** : ハイライトされているとき、クリックして最初の曲線セグメントを選択します。
        - **Select second curve segment** : ハイライトされているとき、クリックしてふたつめの曲線セグメントを選択します。
        - **Radius** : 円の半径を指定します。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
5. 円が作成されます。
