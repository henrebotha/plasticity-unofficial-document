# Control Point Curve

<Reference shortcut="" command="Control Point Curve" button="Toolbar" />

Click on control points to create a curve.

## Basic Usage

1. Click the <ButtonIcon type="button-control-point-curve" /> button on the toolbar.
2. Click to set the starting point of the curve.
3. Click on control points to create the curve.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>K</kbd> **Knife:** Refer to [Using Knife Mode](./sketching-essentials#use-knife-mode).
      - <kbd>Tab</kbd> **Control / Through-point:** Switches to Spline Curve.
      - <kbd>Ctrl + Z</kbd> **Undo:** Cancels the last click.
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrains to X, Y, Z axis](./sketching-essentials#constrain-to-an-axis).
4. After clicking the last control point, right-click to confirm.
5. The curve is created.

<Callout type="info" emoji="ðŸ’¡">
**Difference between two types of curves**
- Spline Curve: Click points through which the curve will pass to create it.
- Control Point Curve: Click on the control points of the curve to create it.
</Callout>
