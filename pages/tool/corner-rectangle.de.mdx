# Corner Rectangle

<Reference shortcut='' command='Corner Rectangle' button='Werkzeugleiste' />

Sie können ein Rechteck erstellen, indem Sie seine Diagonale angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-corner-rectangle" /> Knopf in der Werkzeugleiste.
2. Klicken Sie auf eine Ecke des Rechtecks.
3. Bewegen Sie den Cursor, um die Größe anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](./sketching-essentials#specify-dimensions-numerically).
      - <kbd>K</kbd> **Knife:** Siehe [Messermodus verwenden](./sketching-essentials#use-knife-mode).
      - <kbd>C</kbd> **Center / Corner:** Wechselt zu Mittelpunktrechteck.
      - <kbd>S</kbd> **Square:** Beschränkt auf quadratische Form.
4. Rechtsklick zur Bestätigung.
5. Das Rechteck wird erstellt.
