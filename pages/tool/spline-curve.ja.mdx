# Spline Curve（スプラインカーブ）

<Reference shortcut="" command="Spline Curve" button="ツールバー" />

カーブが通過する点を配置してカーブを作成します。

1. ツールバーのボタン <ButtonIcon type="button-spline-curve" /> をクリックします。
2. スプラインカーブの最初の位置でクリックします。
3. カーブが通過する点をクリックします。クリックを繰り返してスプラインカーブを作成します。
   - <kbd>K</kbd> **Knife:** [ナイフモードを使う](./sketching-essentials#use-knife-mode)を参照してください。
   - <kbd>Tab</kbd> **Control /  Through-point:** Control Point Curveに切り替えます。
   - <kbd>Ctrl + Z</kbd> **Undo:** 直前のクリックを取り消します。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
5. スプラインカーブが作成されます。

<Callout type="info" emoji="💡">
**２つのカーブの違い**
- Spline Curve: カーブが通過する点をクリックして、カーブを作成します。
- Control Point Curve: カーブの制御点をクリックして、カーブを作成します。
</Callout>
