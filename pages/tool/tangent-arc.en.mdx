# Tangent Arc

<Reference shortcut="" command="Tangent Arc" button="Toolbar" />

Create a ark tangent to a curve.

## Basic Usage

1. Click the button <ButtonIcon type="button-tangent-ark" /> on the toolbar.
2. Click to select a curve segments to be tangent to an ark.
3. Specify the radius by moving the cursol.
4. Click to conferm the radius.
6. An arc is created.
