# Three-Point Box

<Reference shortcut='' command='Three-Point Box' button='Werkzeugleiste' />

Sie können ein rechteckiges Feld erstellen, indem Sie drei Punkte auf seiner Basis und seine Höhe angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-three-point-box" /> Knopf in der Werkzeugleiste.
2. Klicken Sie auf eine Ecke der Basis.
3. Klicken Sie auf einen weiteren Punkt auf der Basis.
4. Bewegen Sie den Cursor, um die Größe der Basis anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Befehlsdialog</Kst>
      - **Select target bodies:**
         - Funktioniert wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](/solid/boolean) Befehl.
      - **Width:** Breite angeben.
      - **Length:** Länge angeben.
      - **Height:** Höhe angeben.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Axis:** [Beschränkt auf X, Y, Z Achse](/tool/sketching-essentials#constrain-to-an-axis).
5. Bewegen Sie den Cursor, um die Höhe anzupassen, und klicken Sie zur Bestätigung.
7. Das rechteckige Feld wird erstellt.
