# Center Box

<Reference shortcut='' command='Center Box' button='Werkzeugleiste' />

Ein rechteckiges Feld kann erstellt werden, indem das Zentrum der Basis und die Höhe angegeben werden.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-center-box" /> Knopf in der Werkzeugleiste.
2. Klicken Sie auf den Mittelpunkt der Basis.
3. Bewegen Sie den Cursor, um die Größe der Basis anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](./sketching-essentials#specify-dimensions-numerically).
      - <kbd>C</kbd> **Center/Corner:** Wechselt zu Eckkasten.
      - <kbd>S</kbd> **Square:** Beschränkt auf quadratische Form.
4. Bewegen Sie den Cursor, um die Höhe anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Befehlsdialog</Kst>
      - **Select target bodies**:
         - Gleiche Funktionalität wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](../solid/boolean) Befehl.
      - **Width:** Breite angeben.
      - **Length:** Länge angeben.
      - **Height:** Höhe angeben.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Umschalt + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Umschalt + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>:
         - Gleiche Funktionalität wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](../solid/boolean) Befehl.
      - <kbd>H</kbd> **Height:** Höhe durch Bewegen des Cursors angeben.
5. Sie können auch die Form anpassen, indem Sie die gelben Punkte bewegen.
6. Klicken Sie im Befehlsdialog auf OK oder rechtsklick zur Bestätigung.
7. Das rechteckige Feld wird erstellt.
