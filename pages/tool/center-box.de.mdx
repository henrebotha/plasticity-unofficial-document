# Center Box

<Reference shortcut='' command='Center Box' button='Werkzeugleiste' />

Erstellen Sie ein rechteckiges Kästchen, indem Sie das Zentrum, einen Eckpunkt und die Höhe angeben. Bool'sche Operationen sind ebenfalls verfügbar.

## Grundlegende Nutzung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-center-box" /> in der Werkzeugleiste.
2. Klicken Sie auf den Mittelpunkt der Basisfläche.
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>C</kbd> **Center/Corner:** Wechselt zu Corner Box.
   - <kbd>S</kbd> **Square:** Begrenzt auf ein Quadrat.
3. Klicken Sie am Endpunkt des Umfangs.
4. Bewegen Sie den Cursor und klicken Sie auf die gewünschte Höhe.
5. Legen Sie die Optionen im Befehlsdialog und den Tastenkombinationen fest.
   - <Kst>Befehlsdialog</Kst>
   - **Select target bodies**
      - Diese Optionen haben die gleiche Funktion wie der Befehl Boolean. Weitere Einzelheiten finden Sie im Befehl [**Boolean**](../solid/boolean).
   - **Width:** Geben Sie die Breite an.
   - **Length:** Geben Sie die Länge an.
   - **Height:** Geben Sie die Höhe an.
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>Umschalt + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Umschalt + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - Diese Optionen haben die gleiche Funktion wie der Befehl Boolean. Weitere Einzelheiten finden Sie im Befehl [**Boolean**](../solid/boolean).
   - <kbd>H</kbd> **Height:** Bewegt den Cursor, um die Höhe anzugeben.
6. Bestätigen Sie mit OK im Befehlsdialog oder mit Rechtsklick.
7. Ein rechteckiges Kästchen wird erstellt.
