# Three-Point Arc

<Reference shortcut='' command='Three-Point Arc' button='Werkzeugleiste' />

Sie können einen Bogen erstellen, indem Sie drei Punkte auf seinem Umfang angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-three-point-arc" /> Knopf in der Werkzeugleiste.
2. Klicken Sie auf die erste Position auf dem Umfang des Bogens.
3. Bewegen Sie den Cursor und klicken Sie auf den zweiten Punkt auf dem Umfang.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Axis:** [Beschränkt auf X, Y, Z Achse](/tool/sketching-essentials#constrain-to-an-axis).
4. Bewegen Sie den Cursor und klicken Sie auf den dritten Punkt auf dem Umfang.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
5. Der Bogen wird erstellt.
