# Center Rectangle

<Reference shortcut='' command='Center Rectangle' button='Toolbar' />

Create a rectangle by specifying the center point. Length can be set using mouse actions or with a numerical value.
## Basic Usage

1. Click the button <ButtonIcon type="button-center-rectangle" /> on the toolbar.
2. Click at the point for the rectangle's center.
   - <Kst>Keyboard Shortcuts</Kst>
   - <kbd>Tab</kbd> [Specify length a numerical value](./sketching-essentials#specify-dimensions-numerically).
   - <kbd>K</kbd> **Knife:** [Use Knife Mode.](./sketching-essentials#use-knife-mode) 
   - <kbd>C</kbd> **Center / Corner:** Switches to Corner Rectangle.
   - <kbd>S</kbd> **Square:** Limits to a square.
3. Click at a point on the diagonal.
4. Confirm with OK in the Command Dialog or right-click.
5. The rectangle is created.
