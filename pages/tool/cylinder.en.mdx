# Cylinder

<Reference shortcut='Shift + C' command='Cylinder' button='Toolbar' />

Create a cylinder by specifying a circle's center, diameter, and height. Boolean operations are also available.

1. Click the button <ButtonIcon type="button-cylinder" /> on the toolbar.
2. Click at the point for the circle's center on the base surface.
3. Move the cursor and click on a point on the circumference.
   - <kbd>V</kbd> **Vertical/ Horizontal:** Changes the inclination with respect to the construction plane.
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrain to the X, Y, or Z-axis](./sketching-essentials#constrain-to-an-axis).
4. Move the cursor and click at the desired height.
5. Set the options in the Command Dialog and Keyboard Shortcuts.
   - **Select target bodies**, <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - These options have the same function as the Boolean command. See [**Boolean**](../solid/boolean) command for details.
   - **Radius:** Specify the Radius.
   - **Height:** Specify the Height.
   - <kbd>F</kbd> **Radius:** Move the cursor to specify the radius.
   - <kbd>D</kbd> **Height:** Move the cursor to specify the height.
6. Confirm with OK in the Command Dialog or right-click.
7. A cylinder is created.
