# Regular Polygon（多角形）

<Reference shortcut='' command='Polygon' button='ツールバー' />

任意の数の頂点で多角形を作成できます。

1. ツールバーのボタン <ButtonIcon type="button-regular-polygon" /> をクリックします。
2. 多角形の中心位置でクリックします。
3. カーソルを動かして外接円上の点でクリックします。
   - <kbd>Tab</kbd> [外接円の直径を数値で指定します](./sketching-essentials#specify-dimensions-numerically)。
   - <kbd>Shift</kbd> + ホイール上回転 **Add a Vertex:** 多角形の頂点を増やします。
   - <kbd>Shift</kbd> + ホイール下回転 **Subtract a Vertex:** 多角形の頂点を減らします。
   - <kbd>V</kbd> **Toggle vertical / horizontal :**構築平面に対する傾きを変えます。
   - <kbd>C</kbd> **Toggle circumscribed :** 外接/内接を切り替えます。
   - <kbd>K</kbd> **Knife:** [ナイフモードを使う](./sketching-essentials#use-knife-mode)を参照してください。
   - <kbd>X</kbd>/<kbd>Y</kbd> **X/Y-axis:** [X、Y軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
4. コマンドダイアログのOKをクリックするか、右クリックで確定します。
5. 多角形が作成されます。

<Callout type="info" emoji="💡">
   多角形の頂点は、上下キーでも増減できます。
</Callout>
