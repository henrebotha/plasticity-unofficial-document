# Line

<Reference shortcut="Umschalt + A" command="Line" button="Werkzeugleiste" />

Sie können eine Reihe von verbundenen Liniensegmenten erstellen.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-line" /> Knopf in der Werkzeugleiste.
2. Klicken Sie, um den Startpunkt des Liniensegments festzulegen.
3. Klicken Sie auf den Winkel des Liniensegments, um eine Reihe von Liniensegmenten zu erstellen.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](./sketching-essentials#specify-dimensions-numerically).
      - <kbd>K</kbd> **Knife:** Siehe [Messermodus verwenden](./sketching-essentials#use-knife-mode).
      - <kbd>Strg + Z</kbd> **Undo:** Hebt den letzten Klick auf.
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Axis:** [Beschränkt auf X, Y, Z Achse](./sketching-essentials#constrain-to-an-axis).
4. Nach dem Klicken auf den letzten Punkt, rechtsklick zur Bestätigung.
5. Die Liniensegmente werden erstellt.
