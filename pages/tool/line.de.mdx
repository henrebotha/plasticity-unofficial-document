# Line

<Reference shortcut="Umschalt + A" command="Line" button="Werkzeugleiste" />

Erstellen Sie eine Reihe von verbundenen Liniensegmenten. Geben Sie Länge und Winkel mit Mausaktionen oder mit einem numerischen Wert an.

## Grundlegende Nutzung

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-line" /> in der Werkzeugleiste.
2. Klicken Sie auf den Startpunkt des Liniensegments.
3. Wiederholen Sie das Klicken auf den nächsten Punkt, um die Reihe von Liniensegmenten zu erstellen.
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>Tab</kbd> [Länge und Winkel numerisch angeben](./sketching-essentials#specify-dimensions-numerically).
   - <kbd>K</kbd> **Knife:** [Knife-Modus verwenden.](./sketching-essentials#use-knife-mode)
   - <kbd>Strg + Z</kbd> **Undo:** Den letzten Klick abbrechen.
   - <kbd>X</kbd>/<kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Auf die X-, Y- oder Z-Achse beschränken](./sketching-essentials#constrain-to-an-axis).
4. Bestätigen Sie mit OK im Command Dialog oder mit Rechtsklick.
5. Ein Liniensegment wird erstellt.
