# Three Point Box（3点指定の直方体）

<Reference shortcut='' command='Three-Point Box' button='ツールバー' />

三つの点と高さを指定して立方体を作成できます。ブーリアン操作も可能です。

1. ツールバーのボタン <ButtonIcon type="button-three-point-box" /> をクリックします。
2. 四角形の一辺の始点でクリックします。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
3. 四角形の一辺の終点でクリックします。
4. 四角形の3点目でクリックします。
5. カーソルを動かして目的の高さでクリックします。
6. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - **Select target bodies**, <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - これらのオプションはBooleanコマンドと同じ機能です。詳細は[**Boolean**](../solid/boolean)コマンドを参照してください。
   - **Width:** 幅を指定します。
   - **Length:** 長さを指定します。
   - **Height:** 高さを指定します。
7. 直方体が作成されます。
