# Three Point Box

<Reference shortcut='' command='Three-Point Box' button='ツールバー' />

3点を指定した四角形と高さを指定して直方体を作成できます。

## 基本的な使い方

1. ツールバーのボタン <ButtonIcon type="button-three-point-box" /> をクリックします。
2. 底面の角をクリックします。
3. 底面のもう1点をクリックします。
4. カーソルを動かして底面の大きさを調整し、クリックして確定します。
   - <Kst>コマンドダイアログ</Kst>
      - **Select target bodies :**
         - Booleanコマンドと同じ機能です。詳細は[**Boolean**](/solid/boolean)コマンドを参照してください。
      - **Width:** 幅を指定します。
      - **Length:** 長さを指定します。
      - **Height:** 高さを指定します。
   - <Kst>キーボードショートカット</Kst>
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](/tool/sketching-essentials#constrain-to-an-axis)。
5. カーソルを動かして高さを調整し、クリックして確定します。
7. 直方体が作成されます。
