# Three Point Rectangle

<Reference shortcut='' command='Three Point Rectangle' button='Werkzeugleiste' />

Sie können ein Rechteck erstellen, indem Sie drei Punkte angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-three-point-rectangle" /> Knopf in der Werkzeugleiste.
2. Klicken Sie auf eine Ecke des Rechtecks.
3. Klicken Sie auf einen weiteren Punkt des Rechtecks.
4. Bewegen Sie den Cursor, um die Größe anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](./sketching-essentials#specify-dimensions-numerically).
      - <kbd>K</kbd> **Knife:** Siehe [Messermodus verwenden](./sketching-essentials#use-knife-mode).
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Axis:** [Beschränkt auf X, Y, Z Achse](./sketching-essentials#constrain-to-an-axis).
3. Rechtsklick zur Bestätigung.
5. Das Rechteck wird erstellt.
