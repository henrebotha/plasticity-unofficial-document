# Line

<Reference shortcut="Shift + A" command="Line" button="Toolbar" />

Create a series of connected line segments. Specify length and angle using mouse actions or with a numerical value.

1. Click the button <ButtonIcon type="button-line" /> on the toolbar.
2. Click at the start point of the line segment.
3. Repeat clicking at the next point to create the series of line segments.
   - <kbd>Tab</kbd> [Specify length and angle with a numerical value](./sketching-essentials#specify-dimensions-numerically).
   - <kbd>K</kbd> **Knife:** [Use Knife Mode.](./sketching-essentials#use-knife-mode) 
   - <kbd>Ctrl + Z</kbd> **Undo:** Cancel the last click.
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Constrain to the X, Y, or Z-axis](./sketching-essentials#constrain-to-an-axis).
4. Confirm with OK in the Command Dialog or right-click.
5. A line segment is created.
