# Tangent Arc

<Reference shortcut="" command="Tangent Arc" button="Toolbar" />

カーブに接する円弧を作成できます。

1. ツールバーのボタン <ButtonIcon type="button-tangent-ark" /> をクリックします。
2. 円弧に接するカーブセグメントをクリックして選択します。
3. カーソルを動かして半径を指定します。
4. クリックして半径を確定します。
6. 円弧が作成されます。
