# Corner Box

<Reference shortcut='Shift + B' command='Corner Box' button='Toolbar' />

You can create a rectangular box by specifying the diagonal of its base and its height.

## Basic Usage

1. Click the <ButtonIcon type="button-corner-box" /> button on the toolbar.
2. Click to specify a corner of the base.
3. Move the cursor to adjust the size of the base, and click to confirm.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>Tab</kbd>: Enables direct input of dimensions. See [here](./sketching-essentials#specify-dimensions-numerically) for details.
      - <kbd>C</kbd> **Center/Corner:** Switches to Corner Box.
      - <kbd>S</kbd> **Square:** Restricts to a square base.
4. Move the cursor to adjust the height, and click to confirm.
   - <Kst>Command Dialog</Kst>
      - **Select target bodies**:
         - Functions the same as the Boolean command. Refer to [**Boolean**](../solid/boolean) command for details.
      - **Width:** Specify the width.
      - **Length:** Specify the length.
      - **Height:** Specify the height.
   - <Kst>Keyboard Shortcuts</Kst>
      - <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
         - Functions the same as the Boolean command. Refer to [**Boolean**](../solid/boolean) command for details.
      - <kbd>H</kbd> **Height:** Adjust the height by moving the cursor.
5. You can also adjust the shape by moving the yellow dots.
6. Click OK in the command dialog or right-click to confirm.
7. The rectangular box is created.
