# Center Box（中心からの直方体）

<Reference shortcut='' command='Center Box' button='ツールバー' />

中心、角の点、そして高さを指定して直方体を作成できます。ブーリアン操作も利用可能です。

1. ツールバーのボタン <ButtonIcon type="button-center-box" /> をクリックします。
2. 底面の中心点でクリックします。
   - <kbd>C</kbd> **Center/Corner:** Corner Boxに切り替えます。
   - <kbd>S</kbd> **Square:** 正方形に限定します。
3. 外周の終点でクリックします。
4. カーソルを動かして目的の高さでクリックします。
5. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - **Select target bodies**, <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - これらのオプションはBooleanコマンドと同じ機能です。詳細は[**Boolean**](../solid/boolean)コマンドを参照してください。
   - **Width:** 幅を指定します。
   - **Length:** 長さを指定します。
   - **Height:** 高さを指定します。
   - <kbd>H</kbd> **Height:** カーソルを動かして高さを指定します。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
6. 直方体が作成されます。
