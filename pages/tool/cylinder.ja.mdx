# Cylinder（円柱）

<Reference shortcut='Shift + C' command='Cylinder' button='ツールバー' />

円の中心、直径、そして高さを指定して円柱を作成できます。ブーリアン操作も可能です。

## 基本的な使い方

1. ツールバーのボタン <ButtonIcon type="button-cylinder" /> をクリックします。
2. 円の中心位置でクリックします。
3. カーソルを移動して直径の位置でクリックします。
   - <Kst>キーボードショートカット</Kst>
   - <kbd>V</kbd> **Vertical/ Horizontal:** 構築平面に対する傾きを変更します。
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [X、Y、Z軸に拘束します](./sketching-essentials#constrain-to-an-axis)。
4. カーソルを動かして高さの位置でクリックします。
5. コマンドダイアログとキーボードショートカットでオプションを設定します。
   - <Kst>コマンドダイアログ</Kst>
   - **Select target bodies**
      - Booleanコマンドと同じ機能です。詳細は[**Boolean**](../solid/boolean)コマンドを参照してください。
   - **Radius:** 直径を指定します。
   - **Height:** 高さを指定します。
   - <Kst>キーボードショートカット</Kst>
   - <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>
      - Booleanコマンドと同じ機能です。詳細は[**Boolean**](../solid/boolean)コマンドを参照してください。
   - <kbd>F</kbd> **Radius:** カーソルを動かして直径を指定します。
   - <kbd>D</kbd> **Height:** カーソルを動かして高さを指定します。
5. コマンドダイアログのOKをクリックするか、右クリックで確定します。
6. 円柱が作成されます。
