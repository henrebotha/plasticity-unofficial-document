# Two-Point Circle

<Reference shortcut='' command='Two-Point Circle' button='Werkzeugleiste' />

Erstellen Sie einen Kreis, der durch zwei Punkte des Durchmessers bestimmt wird. Optional können Sie den Winkel relativ zur Konstruktionsebene anpassen.

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-two-point-circle" /> in der Werkzeugleiste.
2. Klicken Sie auf den ersten Punkt am Umfang.
3. Bewegen Sie den Cursor und klicken Sie auf den zweiten Punkt am Umfang.
   - <Kst>Tastenkombinationen</Kst>
   - <kbd>Tab</kbd> : [Den Durchmesser mit einem Zahlenwert angeben](./sketching-essentials#specify-dimensions-numerically).
   - <kbd>V</kbd> **Vertical/Horizontal:** Ändern Sie die Neigung in Bezug auf die Konstruktionsebene.
   - <kbd>K</kbd> **Knife:** [Messermodus verwenden.](./sketching-essentials#use-knife-mode) 
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-axis:** [Einschränken auf die X-, Y- oder Z-Achse](./sketching-essentials#constrain-to-an-axis).
4. Ein Kreis wird erstellt.
