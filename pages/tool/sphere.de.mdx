# Kugel

<Reference shortcut='' command='Sphere' button='Werkzeugleiste' />

Sie können eine Kugel erstellen, indem Sie ihr Zentrum und ihren Durchmesser angeben.

## Grundlegende Nutzung

1. Klicken Sie auf den <ButtonIcon type="button-sphere" /> Knopf in der Werkzeugleiste.
2. Klicken Sie, um das Zentrum der Kugel festzulegen.
3. Bewegen Sie den Cursor, um die Größe der Kugel anzupassen, und klicken Sie zur Bestätigung.
   - <Kst>Befehlsdialog</Kst>
      - **Select target bodies:**
            - Funktioniert wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](/solid/boolean) Befehl.
      - **Radius:** Durchmesser angeben.
   - <Kst>Tastenkombinationen</Kst>
      - <kbd>Tab</kbd>: Ermöglicht direkte Eingabe von Abmessungen. Weitere Informationen finden Sie [hier](/tool/sketching-essentials#specify-dimensions-numerically).
      - <kbd>Umschalt + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Umschalt + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd>:
         - Funktioniert wie der Befehl Boolean. Weitere Informationen finden Sie unter dem [**Boolean**](/solid/boolean) Befehl.
      - <kbd>D</kbd> **Radius:** Durchmesser durch Bewegen des Cursors anpassen.
      - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Axis:** [Beschränkt auf X, Y, Z Achse](/tool/sketching-essentials#constrain-to-an-axis).
4. Sie können auch die Form anpassen, indem Sie den gelben Punkt bewegen.
5. Klicken Sie im Befehlsdialog auf OK oder rechtsklick zur Bestätigung.
6. Die Kugel wird erstellt.
