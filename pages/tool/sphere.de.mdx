# Kugel

<Reference shortcut='' command='Sphere' button='Werkzeugleiste' />

Erstellen Sie eine Kugel, indem Sie deren Mittelpunkt und Durchmesser angeben. Auch boolesche Operationen sind verfügbar.

1. Klicken Sie auf die Schaltfläche <ButtonIcon type="button-sphere" /> in der Werkzeugleiste.
2. Klicken Sie auf den Punkt für das Zentrum der Kugel.
3. Bewegen Sie den Cursor und klicken Sie auf den gewünschten Radius.
4. Legen Sie die Optionen im Befehlsdialog und den Tastenkombinationen fest.
- \[ Befehlsdialog \]
   - **Zielkörper auswählen:** Gleiche Funktion wie der Befehl "Boolean". Siehe [**Boolean**](../solid/boolean) Befehl für Details.
   - **Radius:** Geben Sie den Radius an.
- \[ Tastenkombinationen \]
   - <kbd>Tab</kbd> : [Geben Sie den Durchmesser numerisch an](./sketching-essentials#specify-dimensions-numerically).
   - <kbd>Shift + Q</kbd>, <kbd>Q</kbd>, <kbd>W</kbd>, <kbd>Shift + E</kbd>, <kbd>B</kbd>, <kbd>T</kbd> : 
      - Gleiche Funktion wie der Befehl "Boolean". Siehe [**Boolean**](../solid/boolean) Befehl für Details.
   - <kbd>D</kbd> **Radius:** Bewegen Sie den Cursor, um den Radius zu spezifizieren.
   - <kbd>X</kbd>/ <kbd>Y</kbd>/<kbd>Z</kbd> **X/Y/Z-Achse:** [Beschränken auf die X-, Y- oder Z-Achse](./sketching-essentials#constrain-to-an-axis).
5. Bestätigen Sie mit OK im Befehlsdialog oder mit einem Rechtsklick.
6. Eine Kugel wird erstellt.
