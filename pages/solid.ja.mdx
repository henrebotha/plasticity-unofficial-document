## ソリッドコマンド[#Solid-commands]
| コマンド名                                                   | ショートカット               | 機能                                                                             |
| ------------------------------------------------------------ | ---------------------------- | -------------------------------------------------------------------------------- |
| [Boolean](solid/boolean)                                     | <kbd>Q</kbd>                 | ソリッド、シートをブーリアンする                                                 |
| [Cut](solid/cut)                                             | <kbd>C</kbd>                 | ソリッド、シートをカットする                                                     |
| **ソリッドとシートのコマンド**                               |                              |                                                                                  |
| [Extrude](solid/extrude)                                     | <kbd>E</kbd>                 | フェイス、リージョン、カーブを押し出す                                           |
| [Sweep](solid/sweep)                                         | <kbd>Shift + P</kbd>         | リージョン、カーブをスイープする                                                 |
| [Revolve](solid/revolve)                                     |                              | リージョン、カーブをリボルブする                                                 |
| [Loft](solid/loft)                                           | <kbd>L</kbd>                 | フェイス、エッジ、カーブをロフトする                                             |
| [Pipe](solid/pipe)                                           | <kbd>P</kbd>                 | エッジ、カーブからパイプを作成する                                               |
| [Patch](solid/patch)                                         |                              | シートの穴を埋める<br />閉じたカーブからシートを作成する                         |
| [Thicken](solid/thicken)                                     |                              | シートに厚みをつける                                                             |
| [Thicken Face](solid/thicken-face)                           |                              | フェイスに厚みをつける                                                           |
| [Constrained Surface](solid/constrained-surface)             |                              | 拘束サーフェイスを作成する                                                       |
| [Fillet](solid/fillet)                                       | <kbd>B</kbd>                 | Fillet Curve, Fillet Vertex, Fillet Shell の統一コマンド                         |
| [Fillet Shell](solid/fillet-shell)                           | <kbd>B</kbd>                 | ソリッドまたはシートをフィレットまたは面取りする                                 |
| [Refillet Face](solid/refillet-face)                         |                              | フィレットをやり直す                                                             |
| [Remove Fillets from Shell](solid/remove-fillets-from-shell) |                              | ソリッド、シートのフィレットを削除する                                           |
| [Alternative Duplicate](solid/alternative-duplicate)         | <kbd>Alt + D</kbd>           | 代替複製する                                                                     |
| [Hollow](solid/hollow)                                       |                              | ソリッドを中空にする                                                             |
| [Rebuild](solid/rebuild)                                     |                              | フェイスを再構築する                                                             |
| **シートのコマンド**                                         |                              |                                                                                  |
| [Join](solid/join)                                           | <kbd>J</kbd>                 | Join Curves, Join Sheets の統一コマンド                                          |
| [Join Sheets](solid/join-sheets)                             | <kbd>J</kbd>                 | 一致したエッジでシートをつなぐ                                                   |
| [Bridge Surface](solid/bridge-surface)                       |                              | ふたつのシートをつなぐ                                                           |
| [Extend](solid/extend)                                       |                              | Extend Curve, Extend Sheet の統一コマンド                                        |
| [Extend Sheet](solid/extend-sheet)                           |                              | シートを延長する                                                                 |
| [Unjoin](solid/unjoin)                                       | <kbd>Alt + J</kbd>           | Unjoin Curves, Unjoin Faces, Unjoin Shells の統一コマンド                        |
| [Unjoin Faces](solid/unjoin-faces)                           | <kbd>Alt + J</kbd>           | 選択したフェイスをソリッド、シートから切り離す                                   |
| [Unjoin Shells](solid/unjoin-shells)                         | <kbd>Alt + J</kbd>           | 選択したソリッド、シートからすべてのフェイスを切り離す                           |
| [Find Boundary Edges](solid/find-boundary-edges)             |                              | シートの開口部を選択する                                                         |
| [Untrim](solid/untrim)                                       | <kbd>Alt + T</kbd>           | 元の形状のシートを作成する                                                       |
| **フェイスのコマンド**                                       |                              |                                                                                  |
| [Offset Face](solid/offset-face)                             | <kbd>Shift + O</kbd>         | フェイスをオフセットする                                                         |
| [Draft Face](solid/draft-face)                               |                              | フェイスを傾ける                                                                 |
| [Match Face](solid/match-face)                               |                              | フェイスをほかのフェイスと一致させる                                             |
| [Delete Face](solid/delete-face)                             | <kbd>Shift + Backspace</kbd> | フェイスを削除する                                                               |
| [Unwrap Face](solid/unwrap-face)                             |                              | フェイスをアンラップする                                                         |
| [Wrap Face](solid/wrap-face)                                 |                              | フェイスをラップする                                                             |
| **エッジのコマンド**                                         |                              |                                                                                  |
| [Offset Curve](solid/offset-curve)                           | <kbd>O</kbd>                 | Offset Planar Curve, Offset Region, Offset Face Loop, Offset Edge の統一コマンド |
| [Offset Edge](solid/offset-edge)                             | <kbd>O</kbd>                 | エッジをオフセットして、新しいエッジを作成する                                   |
| [Offset Face Loop](solid/offset-face-loop)                   | <kbd>O</kbd>                 | 選択したフェイスの輪郭をオフセットして、新しいエッジを作成する                   |
| [Imprint](solid/imprint)                                     | <kbd>Shift + I</kbd>         | Imprint Body Body, Imprint Curve Body の統一コマンド                             |
| [Imprint Body Body](solid/imprint-body-body)                 | <kbd>Shift + I</kbd>         | ふたつのソリッドまたは/およびシートの交点にエッジを作成する                      |
| [Imprint Curve Body](solid/imprint-curve-body)               | <kbd>Shift + I</kbd>         | カーブをソリッドまたはシートに投影してエッジを作成する                           |
| [Isoparam](solid/isoparam)                                   | <kbd>Ctrl + R</kbd>          | アイソパラム（エッジ）を加える                                                   |
| [Delete Redundant Topology](solid/delete-redundant-topology) |                              | 余剰なエッジを削除する                                                           |
| [Extend Edge](solid/extend-edge)                             |                              | エッジを延長する                                                                 |
