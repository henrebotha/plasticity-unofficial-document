## Allgemeine Befehle für Skizze und Festkörper
| Befehlsname                                          | Tastenkombination              | Anmerkung                                          |
| ---------------------------------------------------- | ------------------------------ | -------------------------------------------------- |
| [Move](/common/move)                                 | <kbd>G</kbd>                   | Objekte verschieben.                                |
| [Rotate](/common/rotate)                             | <kbd>R</kbd>                   | Objekte drehen.                                     |
| [Scale](/common/scale)                               | <kbd>S</kbd>                   | Objekte skalieren.                                  |
| [Mirror](/common/mirror)                             | <kbd>Alt + X</kbd>             | Objekte spiegeln.                                   |
| [Duplicate](/common/duplicate)                       | <kbd>Umschalt + D</kbd>        | Objekte duplizieren.                                |
| [Place](/common/place)                               | <kbd>Strg + D</kbd>            | Objekte entlang einer bestimmten Achse duplizieren. |
| [Copy with Placement](/common/copy-with-placement)   | <kbd>Strg + Umschalt + C</kbd> | Objekte mit einem Referenzpunkt kopieren.           |
| [Paste with Placement](/common/paste-with-placement) | <kbd>Strg + Umschalt + V</kbd> | Objekte mit einem Referenzpunkt einfügen.           |
| [Rectangular Array](/common/rectangular-array)       |                                | Objekte in einem rechteckigen Muster duplizieren.   |
| [Radial Array](/common/radial-array)                 |                                | Objekte in einem radialen Muster duplizieren.       |
| [Curve Array](/common/curve-array)                   |                                | Objekte entlang einer Kurve duplizieren.            |
| [Set Material](/common/set-material)                 | <kbd>M</kbd>                   | Material setzen.                                    |
| [Remove Material](/common/remove-material)           | <kbd>Alt + M</kbd>             | Material entfernen.                                 |
| [Measure](/common/measure)                           | <kbd>Ctrl + =</kbd>            | Maßwerte messen.                                   |
| [Dimension](/common/dimension)                       | <kbd>Shift + =</kbd>           | Maßwerte ändern.                                   |
|                                                      |                                |                                                    |
