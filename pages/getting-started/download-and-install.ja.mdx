 # ダウンロードとインストール

## ダウンロード
Plasticityには3つのプランがあります： 無料体験版、インディー版、スタジオ版。<br />
無料体験版と他のふたつのプランでは、ダウンロードのプロセスが異なります。
### 無料体験版
Plasticity公式サイトからダウンロードできます。<br />
https://www.plasticity.xyz/

### インディ版とスタジオ版
どちらも公式GitHubのリリースページからダウンロードできます。<br />
https://github.com/nkallen/plasticity/releases

リリースページの例:
![image of relese page ](/img/github-release-page.png)

1. **バージョン**
   - 最新バージョンには「Latest」と表示されています。
   - 「Pre-release」バージョンはスタジオユーザー限定です。

2. **ダウンロードするファイル**
   - ファイル名の末尾を見て、自分のOSに適したファイルをダウンロードしてください。
   - ファイル名をクリックすると、ダウンロードが開始されます。

       | ファイル名                            | OS                      |
       | :------------------------------------- | :---------------------- |
       | Plastycity-0.0.0-.x86_64 **.rpm**      | Linux (rpm) 用          |
       | Plastycity-0.0.0-full **.nupkg**       | NuGet Package Manager用 |
       | Plastycity-0.0.0.setup **.exe**        | Windows用               |
       | Plastycity-darwin- **arm64**-0.0.0.zip | MacOS (M1)用            |
       | Plastycity-darwin- **x64**-0.0.0.zip   | MacOS (Intel)用         |
       | Plastycity_0.0.0_amd64 **.deb**        | Linux (deb)用           |

3. **開発者向けファイル**
   - これらのファイルは開発者向けです。ダウンロードは必要ありません。
