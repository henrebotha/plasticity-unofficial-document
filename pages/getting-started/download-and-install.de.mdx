# Herunterladen und Installierung

Das Verfahren für das Herunterladen und die Installierung variiert je nach Art der Lizenz, die Sie haben. Bitte beziehen Sie sich auf die Anweisungen, die Ihrer Lizenz entsprechen, über die Links unten:
- [Für Benutzer mit Studio-Lizenz](#studio-user)
- [Für Benutzer mit Indie-Lizenz](#indie-user)
- [Für Benutzer, die die kostenlose Testversion nutzen](#trial-user)

## Für Benutzer mit Studio-Lizenz[#studio-user]

Benutzer mit einer Studio-Lizenz können sowohl die Stable Version, die stabil im Betrieb ist, als auch die Beta-Version, die experimentelle Funktionen enthält, verwenden.
Sowohl Stable als auch Beta-Versionen können separat installiert und abwechselnd verwendet werden.

Das gleiche Verfahren gilt sowohl für neue Installierungen als auch für Updates. Eine Deinstallierung der alten Version ist beim Aktualisieren nicht erforderlich.

#### Herunterladen und Installierung
1. Öffnen Sie die offizielle GitHub-Release-Seite.
   - https://github.com/nkallen/plasticity/releases
2. Die neueste Version wird oben auf der Release-Seite angezeigt, mit früheren Versionen unten aufgelistet.
3. Die Version mit der Bezeichnung "Latest" in Grün neben der Versionsnummer ist die neueste Stable Version.
   ![Stable](/img/download-latest.png)
4. Die Version mit der Bezeichnung "Pre-release" in Gelb neben der Versionsnummer ist die Beta-Version.
   ![Beta](/img/download-beta.png)
5. Sie können entweder die Stable- oder die Beta-Version installieren oder beide. Die Reihenfolge der Installierung spielt keine Rolle.
6. Klicken Sie auf die Version, die Sie herunterladen möchten, und gehen Sie zum Dateilistenbildschirm.
<Tabs items={['Windows', 'macOS', 'Linux']}>
   <Tabs.Tab>
   7. Nach dem Klicken auf Assets, um es zu erweitern, klicken Sie auf "plasticity-*****.Setup.exe" aus der Liste der Dateien. Das Herunterladen beginnt.
   8. Sobald das Herunterladen abgeschlossen ist, öffnen Sie die heruntergeladene Datei.
   9. Der Installateur wird gestartet, folgen Sie den Anweisungen, um mit der Installierung fortzufahren.
   </Tabs.Tab>
   <Tabs.Tab>
   7. Nach dem Klicken auf Assets, um es zu erweitern, klicken Sie auf die für die Installierung auf macOS geeignete Datei aus der Liste der Dateien. Das Herunterladen beginnt.
      - Für macOS (M1): Plastycity-darwin-arm64-*****.zip
      - Für macOS (Intel): Plastycity-darwin-x64-*****.zip
   8. Sobald das Herunterladen abgeschlossen ist, entpacken Sie die heruntergeladene Datei.
   9. Ziehen Sie das Anwendungspaket in den "Applications"-Ordner.
   </Tabs.Tab>
   <Tabs.Tab>
   7. Nach dem Klicken auf Assets, um es zu erweitern, klicken Sie auf die geeignete Datei aus der Liste der Dateien. Das Herunterladen beginnt.
      - Für rpm: Plastycity-***.rpm
      - Für NuGet Package Manager: Plastycity-***.nupkg	
      - Für deb: Plastycity_*****.deb
   8. Sobald das Herunterladen abgeschlossen ist, fahren Sie wie gewohnt mit der Installierung fort.
   </Tabs.Tab>
</Tabs>

Nach Abschluss der Installierung fahren Sie mit der Lizenzaktivierung, Radial Menu und Blender Bridge Einrichtung fort.

#### Lizenzaktivierung

Die Lizenzaktivierung ist erforderlich, wenn Sie Plasticity zum ersten Mal verwenden.

10. Starten Sie Plasticity.
11. Klicken Sie auf das P-Menü oben links und klicken Sie auf "Lizenzen verwalten."
   ![P-Menü](/img/p-menu.png)
12. Geben Sie das Token ein und klicken Sie auf "Token validieren."
13. Die Lizenzaktivierung ist abgeschlossen.

#### Radial Menu Einrichtung
14. Eine Einrichtung ist erforderlich, wenn Sie das Radial Menu verwenden. Wenn Sie das Radial Menu nicht verwenden, ist dieser Schritt nicht erforderlich.
15. Für eine Beschreibung der Radial Menu-Funktionen und Einrichtungsanweisungen, bitte beziehen Sie sich auf [hier](/plasticity-essentials/radial-menu).

#### Blender Bridge Einrichtung
16. Eine Add-on Installierung ist erforderlich, wenn Sie die Blender Bridge verwenden. Wenn Sie die Blender Bridge nicht verwenden, ist dieser Schritt nicht erforderlich.
17. Für eine Beschreibung der Blender Bridge-Funktionen und Add-on Installierungsanweisungen, bitte beziehen Sie sich auf [hier](/blender/introduction-blender-bridge).

Die Installierung ist jetzt abgeschlossen.

## Für Benutzer, die die kostenlose Testversion nutzen[#trial-user]

Die kostenlose Testversion kann von der offiziellen Seite heruntergeladen werden.

1. Öffnen Sie die offizielle Seite.
   - https://www.plasticity.xyz/
2. Laden Sie die für Ihr Betriebssystem geeignete Datei herunter.

<Tabs items={['Windows', 'macOS', 'Linux']}>
   <Tabs.Tab>
   3. Sobald das Herunterladen abgeschlossen ist, öffnen Sie die heruntergeladene Datei.
   4. Der Installateur wird gestartet, folgen Sie den Anweisungen, um mit der Installierung fortzufahren.
   </Tabs.Tab>
   <Tabs.Tab>
   3. Sobald das Herunterladen abgeschlossen ist, entpacken Sie die heruntergeladene Datei.
   4. Ziehen Sie das Anwendungspaket in den "Applications"-Ordner.
   </Tabs.Tab>
   <Tabs.Tab>
   3. Sobald das Herunterladen abgeschlossen ist, fahren Sie wie gewohnt mit der Installierung fort.
   </Tabs.Tab>
</Tabs>

Nach Abschluss der Installierung fahren Sie mit der Lizenzaktivierung, Radial Menu und Blender Bridge Einrichtung fort.

#### Lizenzaktivierung
5. Starten Sie Plasticity.
6. Der Lizenzbildschirm wird angezeigt.
7. Klicken Sie auf "Start your free trial."
8. Geben Sie Ihre E-Mail-Adresse und Ihr Passwort ein, und klicken Sie auf "Konto erstellen."
9. Ein Token für die Testversion wird an Ihre E-Mail-Adresse gesendet.
10. Geben Sie das Token ein und klicken Sie auf "Token validieren."
11. Schließen Sie den Lizenzbildschirm. Sie können jetzt die kostenlose Testversion verwenden.

#### Radial Menu Einrichtung
12. Eine Einrichtung ist erforderlich, wenn Sie das Radial Menu verwenden. Wenn Sie das Radial Menu nicht verwenden, ist dieser Schritt nicht erforderlich.
13. Für eine Beschreibung der Radial Menu-Funktionen und Einrichtungsanweisungen, bitte beziehen Sie sich auf [hier](/plasticity-essentials/radial-menu).

#### Blender Bridge Einrichtung
14. Eine Add-on Installierung ist erforderlich, wenn Sie die Blender Bridge verwenden. Wenn Sie die Blender Bridge nicht verwenden, ist dieser Schritt nicht erforderlich.
15. Für eine Beschreibung der Blender Bridge-Funktionen und Add-on Installierungsanweisungen, bitte beziehen Sie sich auf [hier](/blender/introduction-blender-bridge).

Die Installierung ist jetzt abgeschlossen.

### Lizenz erwerben
Wenn Sie Plasticity starten, klicken Sie auf "Buy a license!" auf dem Lizenzbildschirm, um zur Lizenzkaufseite zu gelangen. Sie können auch von der [offiziellen Seite](https://www.plasticity.xyz/#pricing) kaufen.

### Lizenz-Token eingeben
Auf dem Lizenzbildschirm, der angezeigt wird, wenn Sie Plasticity starten, geben Sie das Lizenz-Token in das Token-Eingabefeld auf der rechten Seite ein. Klicken Sie auf "Token validieren", um die Lizenzaktivierung abzuschließen.
