# What's New

## What's New in version 1.3
Release date: October 9, 2023

\[Contents\]


### New features

#### Blender Bridge

Blender Bridgeは、PlasticityからBlenderへのメッシュデータを迅速に転送するための機能です。これはBlenderのアドオンとして実現されています。接続が完了すると、リフレッシュをクリックするだけで、データは非常に迅速にロードされます。一時的なファイルへのエクスポートやすべての設定オプションの調整を心配する必要はありません。

Plasticityでジオメトリを変更しても、Blenderで設定したマテリアルは保持されます。さらに、Plasticityのグループ構造はBlenderのコレクションとして反映されます。この過程で、Plasticityのドキュメント名を識別子としてBlenderの階層が作成されます。そのため、複数のPlasticityドキュメントを1つのBlenderドキュメントに統合することができます。

ライブリンク機能もあります。ライブリンクは本当にクールな機能で、リアルタイムでの更新を見ることができます。Blender Bridgeの真の働き者は、迅速にアセットを取り込む能力であり、それによってBlenderのマテリアル、ライティング、レイトレーシングを使用してアセットを見ることができます。特定のFOVを持つゲームや最終レンダリングで使用される本物のカメラでのシーン内でオブジェクトを見ることができます。Blenderの全機能、エンプティにアイテムを親として設定することを含めて、アニメートしたり、リグを組み立てたり、機械部品がしっかりと合うかどうかを確認することができます。

ワークフローも素晴らしい体験です。BlenderシーンにPlasticityドキュメントを配置した後でも、フェセッティングの設定を変更することができます。PlasticityのほぼすべてのフェセッティングツールはBlender内で利用可能です。これは、モデルに必要なポリゴンの密度やエッジの流れを迅速に取得できる、より速いワークフローを意味します。

現在のバージョンでは、BlenderでのUVシームやシャープの指定は保持されません。しかし、Blenderのアドオンには、エッジを自動的にマークする機能、Plasticityのフェイスやエッジに対応するジオメトリを自動的に選択する機能、およびUVシームをマージする機能が含まれています。これらの機能も、より速いワークフローに貢献しています。

#### Hierarchical isolation

Isolateコマンドの機能が向上し、階層的にオブジェクトを隠すことができるようになりました。これにより、複雑なシーンでの作業が大幅に向上します。Isolateコマンドで表示するオブジェクトを絞り込んだ後、その中からさらにオブジェクトを選択してIsolateコマンドを実行すると、最後に選択したオブジェクトのみが表示されます。この機能は何段階も繰り返すことができます。絞り込んだ段階は画面上に表示され、これをクリックすることで段階的にオブジェクトの表示を戻すことができます。

#### Multiple windows + copy/paste + place command for kitbashing

複数のPlasticityウインドウを開けるようになり作業効率が向上しました。ウィンドウ間でオブジェクトのコピーペーストができますが、さらに便利なCopy with PlacementコマンドとPaste with Placementコマンドが実装されました。これはウインドウをまたぐことができるPlaceコマンドです。最初のウインドウで参照点を持たせてコピーをして、次のウインドウで参照点を活かしてペーストができます。これはキットバッシュにとても便利です。

#### Many enhancements to outliner

- グループフォルダーを右クリックすると、様々な関連コマンドにアクセスできます。例えば、「現在の選択をグループに追加」、「すべて選択」、「現在の選択で新しいサブグループ作成」、すべてのグループを折りたたむ・展開する、素材を変更するなど。

- 選択へスクロール: 3Dビューポートで選択したオブジェクトをアウトライナー内で見つけるためのショートカットが提供されています。

- 選択範囲のみ表示: 選択したオブジェクトが含まれる以外のすべてのグループとセクションを折りたたむショートカットも提供されています。

- 複数のオブジェクトを選択して、一括で状態変更（隠す、無効にする、ロックするなど）や名前の変更、素材の修正ができます。

- グループフォルダーにAltキーを押しながらクリックすると、そのセクション内のすべてのオブジェクトが選択されます。

- セクションラベル（Solids、Sheets、Curves）を右クリックして「すべて選択」を選ぶと、そのセクション内のすべてのオブジェクトが選択されます。

#### Chordal fillets
ソリッドのフィレット機能にChordal オプションが追加されました。Chordalオプションの特長は、フィレットされる角度に関係なく、フィレットの弦の長さを一定に保つ点です。この一貫性は、モデルの異なる部分で視覚的な整合性を保つ際に特に有益です。
